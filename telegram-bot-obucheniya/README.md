# Telegram-бот для обучения агентов СРЕДА

Образовательный Telegram-бот для агентов по продажам новостроек. Позволяет обучать агентов, отслеживать прогресс, бронировать объекты и управлять командой.

## 🚀 Основные возможности

### Для агентов:
- 📚 Структурированное обучение по 7 блокам
- 📊 Отслеживание личного прогресса
- 🎯 Тестирование знаний
- 🏠 Система бронирования новостроек
- 💎 Доступ к премиум-контенту

### Для администраторов:
- 👥 Управление агентами
- 📊 Аналитика прогресса
- 📣 Рассылки (квартира недели, напоминания)
- 🏠 Контроль бронирований
- 💰 Управление платежами

## 📦 Установка

### 1. Клонируйте репозиторий
```bash
git clone <repository-url>
cd telegram-bot-obucheniya
```

### 2. Установите зависимости
```bash
pip install -r requirements.txt
```

### 3. Настройте конфигурацию
Скопируйте шаблон `.env` и заполните его:
```bash
cp .env.example .env
```

Заполните в файле `.env`:
- `BOT_TOKEN` - токен вашего бота от [@BotFather](https://t.me/BotFather)
- `ADMIN_IDS` - ID администраторов через запятую (узнать через [@userinfobot](https://t.me/userinfobot))
- `GEMINI_API_KEY` - ключ Gemini API для ИИ-проверки ответов (опционально)
- `PAYMENT_PROVIDER_TOKEN` - токен платежной системы (опционально)

### 4. Запустите тесты
```bash
python3 test_bot.py
```

### 5. Запустите бота
```bash
python3 bot.py
```

## 🏠 Система бронирования

Агенты могут бронировать новостройки для клиентов прямо в боте:

1. Выбор новостройки из списка
2. Ввод информации о квартире
3. Указание данных клиента
4. Создание бронирования

Администратор видит все бронирования и может контролировать процесс.

## 🚀 Развёртывание

### Локальный запуск:
```bash
python3 bot.py
```

### Запуск в фоновом режиме:
```bash
nohup python3 bot.py > bot.log 2>&1 &
```

### Через systemd (Linux):
1. Создайте файл сервиса: `/etc/systemd/system/telegram-bot.service`
2. Добавьте конфигурацию:
```ini
[Unit]
Description=Telegram Bot for Agents Training
After=network.target

[Service]
Type=simple
User=your_user
WorkingDirectory=/path/to/bot
ExecStart=/usr/bin/python3 bot.py
Restart=always

[Install]
WantedBy=multi-user.target
```
3. Запустите сервис:
```bash
sudo systemctl enable telegram-bot
sudo systemctl start telegram-bot
```

## 🛠️ Техническая поддержка

### Возможные проблемы:

1. **Бот не запускается**
   - Проверьте токен в `.env` файле
   - Убедитесь, что все зависимости установлены
   - Запустите тест: `python3 test_bot.py`

2. **База данных не создается**
   - Проверьте права на запись в директорию
   - Убедитесь, что установлен aiosqlite

3. **Ошибки импорта**
   - Проверьте, что вы в правильной директории
   - Убедитесь, что все файлы на месте

### Логирование:
Все события логируются в консоль. Для сохранения логов в файл используйте:
```bash
python3 bot.py >> bot.log 2>&1
```
