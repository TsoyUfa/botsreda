# Выводы после сессий

> Ключевые инсайты после созвонов, анализа, встреч. Не черновики - а конкретные выводы: что поняли, что меняем, как применяем. Если нужен сырой материал - он в `raw/`.

## Формат файла

Каждый вывод - отдельный файл с датой: `YYYY-MM-DD-краткое-название.md`

```
# Выводы: [тема/повод]

**Дата:** ДД.ММ.ГГГГ
**Контекст:** созвон с X / анализ Y / встреча по Z

## Что поняли
- [инсайт 1]
- [инсайт 2]

## Что меняем
- [конкретное действие]

## Как применяем
- [следующий шаг]
- [кто делает]
- [дедлайн]

## Связанные файлы
- [[ссылка на проект/стратегию/клиента]]
```

## Правила

- Один файл = одна сессия/событие
- Писать сразу после события, пока свежо
- Если инсайт относится к клону (стиль, голос) - продублировать в `3. Мой клон/feedback/`
- Если инсайт относится к мастерству - продублировать в `4. Мастерство/` (соответствующая линза)
