# Журнал решений

> Хронология всех значимых бизнес-решений. Каждая запись - почему решили именно так, чтобы через 3-6 месяцев не вспоминать «а почему мы тогда отказались от клиента X?».

## Формат записи

```
## [ДД.ММ.ГГГГ] Решение: [кратко]
- **Контекст:** что происходило
- **Варианты:** что рассматривали
- **Выбрали:** что решили
- **Почему:** обоснование
- **Инициировал:** Антон / партнёр / агент
- **Связанные файлы:** [[ссылки на проекты, стратегии, клиентов]]
```
