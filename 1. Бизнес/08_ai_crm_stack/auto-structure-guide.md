# Руководство по автоматизации "второго мозга" с Hermes и Obsidian

## Обзор системы

Вы создали мощную систему автоматизации для управления знаниями, которая:
- Автоматически распознаёт тип информации
- Распределяет заметки по квадрантам РОСТ
- Интегрируется с голосовым вводом через Telegram
- Сохраняет важные сессии и решения
- Обеспечивает целостность вашего Knowledge Graph

## Компоненты системы

### 1. Навык `auto-structure-notes`
**Файл:** `business/auto-structure-notes/SKILL.md`

Основной навык для автоматической структуризации заметок:
- Распознаёт 4 типа контента: Бизнес, План, Клон, Мастерство
- Определяет подходящие подпапки для размещения
- Создает связи между заметками
- Интегрируется с голосовым вводом

### 2. Скрипт `structure-notes.py`
**Файл:** `scripts/structure-notes.py`

Программная реализация системы структуризации:
- Анализирует текст и определяет тип контента
- Генерирует имена файлов в стандартном формате
- Создает заметки с правильной структурой
- Работает из командной строки

**Использование:**
```bash
# Создать заметку с автоматическим определением типа
python3 scripts/structure-notes.py "Текст заметки" --title "Заголовок"

# Указать тип вручную
python3 scripts/structure-notes.py "Текст" --title "Заголовок" --type business
```

### 3. Скрипт `save-sessions.sh`
**Файл:** `scripts/save-sessions.sh`

Автоматическое сохранение важных сессий Hermes:
- Создает файлы сессий в папке `2. План/conclusions/`
- Обновляет дашборд и журнал решений
- Работает как cron-задача

### 4. Интеграция с Telegram
**Файл:** `1. Бизнес/08_ai_crm_stack/telegram-voice-integration.md`

Система для голосового ввода заметок:
- Telegram-бот с командами `/note`, `/business`, `/plan`, `/clone`, `/mastery`
- Распознавание речи и автоматическая структуризация
- Уведомления о создании заметок

### 5. Cron-задачи
**Файл:** `crontab.txt`

Автоматические задачи:
- Ежедневное сохранение сессий (9:00 и 20:00)
- Ежечасная проверка новых заметок
- Обновление индекса vault
- Резервное копирование

Выберите удобный способ:

```

```

## Примеры работы системы

5. Создаёт связи: с `03_audiences/agents.md`

**Вход:** "Я всегда начинаю день с проверки дашборда и ответа на важные сообщения, это мой главный принцип"

**Действия системы:**

3. Создаёт файл: `2026-06-18_clone_daily-dashboard-principle.md`
4. Добавляет в: `3. Мой клон/principles/`
5. Связывает с: `3. Мой клон/voice/tone.md`

### Пример 3: Методика
**Вход:** "Разработал фреймворк 3-ступенчатой квалификации клиента: 1) Бюджет, 2) Сроки, 3) Готовность"

**Действия системы:**

## Частые вопросы

### Q: Как изменить тип контента для заметки?
A: Откройте файл заметки и измените поле "Тип:" в метаданных. Система запомнит ваш выбор.

### Q: Как создать связи между заметками?
A: В тексте заметки используйте формат `[[Название заметки]]`. Система автоматически создаст обратные ссылки.

### Q: Как добавить новую подпапку?
A: Создайте папку в соответствующем квадранте и добавьте её в список в навыке `auto-structure-notes`.

### Q: Как отключить автоматическое сохранение?
A: Удалите соответствующую строку в `crontab.txt` и обновите cron: `crontab crontab.txt`.
