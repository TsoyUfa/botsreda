# Фильтры бизнес-решений

## MVP Money Filter

Перед новым продуктом, оффером или большим планом ответить:

1. **Кто платит?** Застройщик, РОП, агентство, банк, покупатель, партнёр?
2. **За что платит?** Аудит, внедрение, обучение, сопровождение, SaaS, сделка?
3. **Почему сейчас?** Где боль: продажи, CRM, партнёрка, ипотека, хаос, скорость?
4. **Как проверить за 7 дней?** 3 созвона, демо, мини-аудит, КП, пилот?
5. **Что НЕ делаем в MVP?** Что отсекаем, чтобы не утонуть?

## Оценка идеи

| Критерий | /5 | Вопрос |
|---|---:|---|
| Путь к 500к+ |  | Ведёт ли к 500к+ ₽/мес? |
| Скорость запуска |  | Можно ли проверить за 7-14 дней? |
| Масштабируемость |  | Можно ли повторить методологией/шаблоном? |
| Низкое сопротивление |  | Есть ли тёплый доступ к клиенту? |
| Win-win партнёров |  | Выгодно ли всем сторонам? |

## Правило отсечения

Если нет понятного платящего клиента, срочной боли и проверки за 7 дней - идея уходит в бэклог.

## Правило снижения важности

Если задача раздулась, сократить до одного действия на 24 часа: кому написать, что предложить, какой результат получить.
