# Аудитория: агенты и агентства

## Что им важно

- понятный продукт застройщика;
- быстрый ответ менеджера;
- прозрачная комиссия;
- отсутствие конфликтов по клиенту;
- материалы, которые помогают объяснять объект;
- обучение без воды;
- инструменты для личного бренда и привлечения клиентов.

## Боли

- клиент перегружен и боится ошибиться;
- сложно объяснять ипотеку и сравнивать ЖК;
- застройщик долго отвечает;
- непонятно, кому объект подходит;
- агент не хочет рисковать репутацией;
- нет системы follow-up.

## Что им дать

- карта первой встречи;
- чек-лист диагностики;
- карточка объекта;
- ответы на возражения;
- сценарии follow-up;
- контент-план;
- канал быстрой связи с застройщиком.
