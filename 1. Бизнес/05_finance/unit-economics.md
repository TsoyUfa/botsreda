# Юнит-экономика

## Ретейнер партнёрского канала

| Показатель | Значение |
|---|---:|
| Цена | 150-200к ₽/мес |
| Минимальный срок | 3-6 месяцев |
| LTV при 6 мес × 200к | 1,2 млн ₽ |
| Прямые расходы | 50-100к ₽/мес на мероприятия/помощников |
| Маржинальность | 50-70% |
| Ограничение | максимум 3 клиента одновременно |

## Экспресс-аудит

| Показатель | Значение |
|---|---:|
| Цена | 25-50к ₽ |
| Срок | 3-5 дней |
| Роль | вход в воронку и сбор кейсов |
| Апселл | стратсессия / ретейнер |

## AI/CRM-внедрение

| Показатель | Значение |
|---|---:|
| Внедрение | 150-300к ₽ |
| Поддержка | 30-100к ₽/мес |
| Главный риск | сложность интеграции и доступ к данным |
| MVP | анализ ограниченной выборки звонков/CRM |

## Правило

Считать экономику в рублях, с учётом времени Антона и нагрузки на сопровождение.
