# Финансовый путь к 500к+ ₽/мес

## Цель

Стабильный доход 500к+ ₽/мес через B2B-продукты с высоким чеком и повторяемой методологией.

## Минимальная конфигурация

```text
2 ретейнера × 200к = 400к
2 стратсессии × 50к = 100к
Итого = 500к ₽/мес
```

## Флагманская конфигурация

```text
3 ретейнера × 200к = 600к ₽/мес
```

## Расширенная конфигурация

```text
3 ретейнера = 600к
2 завтрака/мероприятия = 200к
2 стратсессии = 100к
Итого = 900к ₽/мес
```

## Самый короткий путь

Не начинать с платформы. Начать с платных диагностик и ретейнерного консалтинга, где текущие связи и экспертиза дают максимальное плечо.
