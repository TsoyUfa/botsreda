## Критерии приоритета

1. Есть личный контакт или тёплый вход.
2. Есть партнёрский канал.
3. Есть видимый хаос/потери в агентской работе.
4. Есть бюджет 150-200к ₽/мес.
5. Можно быстро провести диагностику.

## Следующее действие

Для каждого аккаунта сделать карточку: ЛПР, текущая боль, какой оффер предложить, дата касания, следующий шаг.
