# Воронки продаж

## Главная B2B-воронка

```text
Тёплый контакт / рекомендация
-> Бесплатная диагностика 30 минут
-> Экспресс-аудит 25-50к ₽
-> Стратсессия 50-80к ₽
-> Ретейнер 150-200к ₽/мес
-> AI/CRM-допродажи
```

## Воронка через аудит

1. 10 персональных сообщений.
2. 5-7 диагностик.
3. 3-5 платных аудитов.
4. 1-2 индивидуальных КП.
5. 1 ретейнер.

## Воронка через обучение

```text
Проблема: агенты плохо продают объект
-> обучение/интенсив
-> обратная связь от агентского рынка
-> настройка партнёрского канала
-> ретейнер / амбассадорская программа
```

## Воронка через AI/CRM

```text
Боль РОПа: не вижу звонки и CRM
-> мини-аудит 20-50 звонков/сделок
-> отчёт: где теряются деньги
-> внедрение AI-аналитики
-> ежемесячная аналитика / ассистент РОПа
```
