# Оффер: партнёрская сеть застройщика под ключ

## Коротко

Антон выстраивает для застройщика работающий партнёрский канал с агентствами: диагностика, ABC-сегментация, отношения с агентствами, мероприятия, регламенты, SLA, обучение команды и ежемесячная поддержка.

## Позиционирование

**Личный стратег по партнёрскому каналу для застройщика.** Не консультант, который даёт советы, а внешний партнёр, который делает работу: встречается с агентствами, выстраивает отношения и помогает довести партнёрский канал до продаж.

## Для кого

- застройщики с объёмом 50+ квартир в год;
- есть или планируется партнёрский канал;
- РОП/коммерческий директор понимает ценность агентств;
- есть бюджет 150-200к ₽/мес;
- готовы работать минимум 3-6 месяцев.

## Главные боли

- агентств в базе много, активных мало;
- заявки теряются, ответы долгие;
- агенты продают конкурентов, потому что там проще;
- нет выделенного человека на партнёрский канал;
- РОП перегружен операционкой;
- нет регулярной обратной связи с рынка.

## Что входит

1. Диагностика партнёрской сети.
2. ABC-сегментация агентств.
3. Карта коммуникаций и SLA.
4. Личные встречи с ключевыми агентствами.
5. Мероприятия/завтраки/обучение для агентов.
6. Материалы для продаж объекта.
7. Еженедельный созвон с РОПом.
8. Ежемесячный отчёт и корректировка стратегии.

## Обещание результата

Системная работа с 30-50 агентствами, рост активности партнёрского канала и потенциальный прирост продаж на 15-30%+ при нормальной продуктовой базе.
