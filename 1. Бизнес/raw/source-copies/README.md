# Source copies

Здесь лежат копии ключевых исходных заметок, из которых собрана папка `1. Бизнес`.

Оригиналы не удалялись и остались на своих местах. Копии нужны, чтобы бизнес-контекст был рядом с рабочими сводками и чтобы агент мог быстро найти первоисточник.

## Правило

Если исходник длинный, не тащить его целиком в рабочие файлы. Сначала сделать короткую выжимку: продукт, аудитория, боль, оффер, цена, следующий шаг.
