# 🚀 Telegram-Бот Обучения «Эксперт Сити»

Асинхронный Telegram-бот на Python (aiogram 3.x) для обучения агентов и клиентов первичному рынку недвижимости (новостройкам) с разделением на бесплатный базовый блок и платные профессиональные модули, интерактивным тестированием и встроенной оплатой.

---

## 🛠 Установка зависимостей

Для запуска бота требуется Python версии **3.9+**. 

Установите необходимые библиотеки:
```bash
pip install aiogram aiosqlite
```

---

## ⚙️ Настройка и Конфигурация

Вы можете настроить бота через переменные окружения или отредактировать файл [config.py](file:///Users/anton_tsoy/Desktop/Обсидиан/telegram-bot/config.py):

1. **Токен бота (Bot Token):** Получите у `@BotFather` и укажите в `config.py` или задайте через терминал:
   ```bash
   export BOT_TOKEN="123456789:ABCdefGhIJKlmNoPQRsTUVwxyZ"
   ```
2. **Токен платежного провайдера (необязательно):** Для приема платежей банковскими картами (ЮKassa/Robokassa) введите токен провайдера, полученный в `@BotFather`:
   ```bash
   export PAYMENT_PROVIDER_TOKEN="123456789:TEST:12345"
   ```
   *Если токен провайдера пуст, бот автоматически переключится в **тестовый режим**, подтверждая оплаты без реального списания средств!*
   *Для Telegram Stars токен не требуется, валюта звезд `XTR` работает из коробки.*

---

## 🚀 Запуск бота

Запустите скрипт бота из терминала:
```bash
python bot.py
```

При первом запуске бот автоматически создаст базу данных SQLite `expert_city_bot.db` во временной папке и начнет опрашивать сервера Telegram (Long Polling).

---

## 📂 Структура проекта

* `bot.py` — Главный исполняемый файл. Маршрутизация команд, стейт-машина тестов (FSM), обработка счетов и платежей.
* `config.py` — Файл настроек: цены, блоки обучения, база данных.
* `db.py` — Асинхронный интерфейс для взаимодействия с SQLite базой данных (`aiosqlite`).
* `lessons.py` — База данных текстов уроков, аудиоподкастов и вопросов для интерактивных тестов.
