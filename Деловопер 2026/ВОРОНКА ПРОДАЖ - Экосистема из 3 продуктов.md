- **Выход:** 10 диагностик
- **Конверсия:** 50%

### Действия:

**Источники:**
1. Личные связи (ТОП-20 застройщиков Уфы)
2. Рекомендации от существующих клиентов
3. Социальные сети (LinkedIn, Telegram)
4. Мероприятия и конференции

**Список целевых застройщиков:**
- Третий Трест
- Первый Трест
- Prime Development
- СУ №1
- АРХСТРОЙ ГРУПП
- НОВЫЙ ДОМ
- БРИГ
- Садовое кольцо
- Жилой Квартал
- Самолет
- [Добавить ещё 10]

#### Шаг 1.2: Первый контакт

**Формат:** Telegram/WhatsApp сообщение

**Скрипт:**
```
[Имя], привет!

Хочу предложить тебе кое-что интересное.

Я последние 3 года занимаюсь только одним — 
выстраиваю партнёрские сети для застройщиков.

Беру любого застройщика (от мелкого до крупного) 
и строю работающую сеть с агентствами. Не просто 
процессы, а ОТНОШЕНИЯ.

У меня есть прямой доступ ко всем агентствам города, 
я знаю, как заставить их продавать именно твои объекты.

Работаю максимум с 3 застройщиками одновременно. 
Сейчас освобождается 1 место.

Хочу показать, как это работает. 30 минут созвона — 
покажу конкретно для [Название компании].

Завтра/послезавтра свободен?
```

**Follow-up:**
- Если нет ответа через 24 часа — напомнить
- Если нет ответа через 48 часов — ещё одно сообщение
- Если нет ответа через 72 часа — закрыть лид

#### Шаг 1.3: Предложение диагностики

**Формат:** После первого контакта

**Скрипт:**
```
[Имя], отлично, что заинтересовался!

Предлагаю начать с бесплатной диагностики 
партнёрского канала (30 минут).

Что сделаю:
✅ Проанализирую текущую ситуацию
✅ Покажу 3-5 ключевых проблем
✅ Дам рекомендации, что делать дальше

Это бесплатно и без обязательств. Просто 
хочу показать ценность.

Когда удобно? Могу сегодня/завтра в [время].
```

#### Шаг 1.4: Проведение диагностики

**Формат:** Онлайн-встреча (30 минут)

**Структура:**
1. Вопросы для понимания ситуации (10 минут)
2. Презентация проблем (10 минут)
3. Предложение аудита (10 минут)

**Результат:** Понимание проблем, доверие, предложение аудита

## 🎯 ОПТИМИЗАЦИЯ ВОРОНКИ

### Точки оптимизации:

1. **Конверсия диагностика → аудит:**
   - Текущая: 50%
   - Цель: 60%
   - Действия: Улучшить скрипты, добавить кейсы

2. **Конверсия аудит → аналитика:**
   - Текущая: 40%
   - Цель: 50%
   - Действия: Показать ценность данных, добавить примеры

3. **Конверсия аналитика → партнёрская сеть:**
   - Текущая: 40%
   - Цель: 50%
   - Действия: Показать синергию, добавить ROI-калькулятор

4. **Конверсия партнёрская сеть → ИИ-канал:**
   - Текущая: 60%
   - Цель: 70%
   - Действия: Показать экономию времени, добавить демо

### Автоматизация:

1. **CRM для отслеживания лидов:**
   - Статус каждого лида
   - Следующие шаги
   - Напоминания

2. **Шаблоны сообщений:**
   - Первый контакт
   - Follow-up
   - Предложение диагностики

3. **Автоматические напоминания:**
   - Когда звонить клиенту
   - Когда отправлять материалы
   - Когда делать апселл
