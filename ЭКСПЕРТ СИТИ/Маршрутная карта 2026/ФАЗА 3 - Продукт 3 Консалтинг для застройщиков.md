# ФАЗА 3: ПРОДУКТ 3 - КОНСАЛТИНГ ДЛЯ ЗАСТРОЙЩИКОВ

**Срок:** Месяцы 5-6 (Май - Июнь 2026)  
**Статус:** Запланировано  
**Приоритет:** Высокий

---

## ЦЕЛЬ ФАЗЫ

Создать консалтинговый продукт для застройщиков по ипотечным продуктам и агентскому маркетингу, что позволит выбивать лучшие условия для агентств.

**Ключевые результаты:**
- 1-2 застройщика на консалтинге
- Реализованные ипотечные продукты
- Настроенный агентский маркетинг
- Специальные условия для агентств-партнеров (комиссии, скидки, приоритет)

---

## МЕСЯЦ 5: РАЗРАБОТКА ПРОДУКТОВ ДЛЯ ЗАСТРОЙЩИКОВ

### Неделя 1: Исследование потребностей застройщиков

**Задачи:**
- [ ] Провести интервью с 3-5 застройщиками
  - Какие проблемы с ипотекой?
  - Как работает агентский канал сейчас?
  - Какие улучшения нужны?
  - Какие результаты ожидают?
  
- [ ] Изучить текущие проблемы с ипотекой и агентским каналом
  - Анализ текущих ипотечных программ
  - Проблемы с банками
  - Низкая конверсия агентского канала
  - Отсутствие системы работы с агентствами
  
- [ ] Определить ключевые продукты для предложения
  - Ипотечные продукты (оптимизация программ, работа с банками)
  - Агентский маркетинг (развитие канала, обучение, материалы)
  - Специальные условия для агентств (комиссии, скидки)
  
- [ ] Создать структуру консалтингового предложения
  - Пакеты услуг (Старт, Рост, Премиум)
  - Что входит в каждый пакет
  - Сроки реализации
  - Ожидаемые результаты

**Результат:** Понимание потребностей и структура предложения

---

### Неделя 2: Ипотечные продукты

**Задачи:**
- [ ] Разработать методологию работы с ипотечными продуктами
  - Анализ текущих программ застройщика
  - Сравнение с конкурентами
  - Оптимизация условий
  - Работа с банками-партнерами
  
- [ ] Создать шаблоны предложений для банков
  - Структура предложения
  - Обоснование выгод для банка
  - Условия сотрудничества
  - Метрики успеха
  
- [ ] Подготовить материалы по оптимизации ипотечных программ
  - Рекомендации по ставкам
  - Программы государственной поддержки
  - Специальные условия для покупателей
  - Интеграция с продажами
  
- [ ] Разработать систему мониторинга ипотечных ставок
  - Отслеживание ставок конкурентов
  - Анализ рынка
  - Рекомендации по корректировке
  - Автоматические уведомления

**Результат:** Полная методология работы с ипотечными продуктами

---

### Неделя 3: Агентский маркетинг

**Задачи:**
- [ ] Разработать методологию развития агентского канала
  - ABC-анализ агентств
  - Стратегия работы с каждым сегментом
  - План развития канала
  - Метрики эффективности
  
- [ ] Создать материалы для работы с агентствами
  - Презентации проектов
  - Брошюры и каталоги
  - Онлайн-материалы
  - Обучение агентов
  
- [ ] Подготовить систему обучения агентов застройщика
  - Программа обучения
  - Материалы и презентации
  - Формат (онлайн/офлайн)
  - Сертификация
  
- [ ] Разработать регламенты взаимодействия
  - Регламент работы с агентствами
  - Процесс закрепления клиентов
  - Система комиссий
  - Разрешение конфликтов

**Результат:** Полная методология агентского маркетинга

---

### Неделя 4: Упаковка продукта

**Задачи:**
- [ ] Создать презентацию консалтингового предложения
  - Проблемы застройщиков
  - Решения и методологии
  - Кейсы и результаты
  - Пакеты и ценообразование
  
- [ ] Подготовить кейсы и примеры
  - Кейсы из предыдущего опыта
  - Примеры результатов
  - Отзывы клиентов
  - Метрики успеха
  
- [ ] Разработать ценообразование
  - Стоимость пакетов
  - Формат оплаты (ретейнер/разово)
  - Условия и гарантии
  - Дополнительные услуги
  
- [ ] Создать договоры и регламенты
  - Шаблоны договоров
  - Регламенты работы
  - KPI и SLA
  - Отчетность

**Результат:** Готовый консалтинговый продукт для застройщиков

## МЕСЯЦ 6: ЗАПУСК И ПЕРВЫЕ КЛИЕНТЫ

### Неделя 1-2: Поиск первых клиентов

**Задачи:**
- [ ] Составить список целевых застройщиков (10-15 компаний)
  - Активные застройщики в регионе
  - Застройщики с проблемами в ипотеке/агентском канале
  - Застройщики с потенциалом роста
  - Личные связи и рекомендации
  
- [ ] Провести холодные контакты
  - Подготовка персональных предложений
  - Первые контакты (звонки, письма, встречи)
  - Презентация продукта
  - Ответы на возражения
  
- [ ] Провести презентации продукта
  - Демонстрация методологий
  - Показ кейсов
  - Обсуждение потребностей
  - Предложение решений
  
- [ ] Заключить первые договоры (цель: 1-2 клиента)
  - Согласование условий
  - Подписание договоров
  - Планирование старта работ
  - Подготовка команды

**Результат:** 1-2 застройщика на консалтинге

  - Создание материалов
  - Запуск обучения
  
- [ ] Выбить специальные условия для агентств-партнеров
  - Переговоры с застройщиком
