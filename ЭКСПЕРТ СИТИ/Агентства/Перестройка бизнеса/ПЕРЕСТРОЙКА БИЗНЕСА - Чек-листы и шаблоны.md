## ИНСТРУКЦИИ ПО ИСПОЛЬЗОВАНИЮ

### Как использовать чек-листы

1. **Распечатайте чек-листы** или используйте в цифровом виде
2. **Отмечайте выполненные задачи** по мере выполнения
3. **Регулярно проверяйте прогресс** (ежедневно/еженедельно)
4. **Корректируйте при необходимости** в зависимости от ситуации

### Как использовать шаблоны

1. **Скопируйте шаблон** в новый документ
2. **Заполните все поля** [X], [Y], [Z] реальными данными
3. **Адаптируйте под вашу ситуацию** при необходимости
4. **Используйте регулярно** для стандартизации процессов
