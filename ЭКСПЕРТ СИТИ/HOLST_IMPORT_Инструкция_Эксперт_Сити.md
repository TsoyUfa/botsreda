# HOLST IMPORT — инструкция для доски «Эксперт Сити»

## Что это

Этот файл объясняет, как перенести стратегическую карту `Эксперт Сити` в Holst.

Главные файлы для переноса:

- `HOLST_IMPORT_Карточки_Эксперт_Сити.csv` — короткие карточки для доски.
- `MIRO_IMPORT_Стратегическая_карта_Эксперт_Сити.md` — полная карта фреймов и смыслов.
- `MIRO_IMPORT_Карточки_Эксперт_Сити.csv` — исходная CSV-версия, совместимая с таблицами.

## Важно про Holst

У Holst есть полноценный импорт из Miro через `.holst` backup-файл. Этот файл создаётся расширением Holst из уже готовой Miro-доски.

Я не генерирую `.holst` напрямую, потому что это внутренний формат Holst. Зато подготовлена структура, которую можно быстро перенести двумя способами.

---

# Вариант 1 — быстро, напрямую в Holst

Используйте этот вариант, если вам нужна рабочая доска без промежуточного Miro.

## Шаг 1. Создайте новую доску

Название:

`Эксперт Сити — стратегическая карта`

## Шаг 2. Создайте 9 фреймов

Разместите фреймы слева направо или сеткой 3x3:

1. `01. Обзор проекта`
2. `02. Экосистема продуктов`
3. `03. Квартальный фокус`
4. `04. Дорожная карта`
5. `05. Продажи и деньги`
6. `06. Оргструктура и роли`
7. `07. Академия и обучение`
8. `08. Ребрендинг и упаковка`
9. `09. Реестр источников`

## Шаг 3. Импортируйте карточки

Откройте файл:

`HOLST_IMPORT_Карточки_Эксперт_Сити.csv`

Колонки:

- `HolstFrame` — в какой фрейм положить карточку.
- `Order` — порядок внутри фрейма.
- `ObjectType` — тип: фрейм или карточка.
- `CardTitle` — название карточки.
- `CardText` — текст карточки.
- `Priority` — приоритет.
- `Owner` — владелец.
- `Source` — исходный файл в Obsidian.

Если Holst не импортирует CSV напрямую, откройте CSV в Numbers/Excel/Google Sheets и копируйте строки блоками по фреймам.

## Шаг 4. Цветовая логика

Рекомендуемые цвета:

- `High` — красный или оранжевый.
- `Medium` — жёлтый.
- `Low` — серый.
- `ObjectType = Frame` — синий или тёмный заголовок.

## Шаг 5. Логика работы с доской

Используйте доску не как архив, а как панель управления:

- Фрейм 1 — объяснить проект за 3 минуты.
- Фрейм 2 — показать, как продукты связаны.
- Фрейм 3 — вести квартальное исполнение.
- Фрейм 5 — обсуждать деньги и путь к 500К+ ₽/мес.
- Фрейм 9 — быстро открывать исходники в Obsidian.

# Вариант 2 — максимально точный импорт через Miro

Используйте этот вариант, если нужен именно импорт в Holst с сохранением объектов.

## Шаг 1. Соберите доску в Miro

Используйте файл:

`MIRO_IMPORT_Стратегическая_карта_Эксперт_Сити.md`

Создайте фреймы и карточки по структуре файла.

## Шаг 2. Скачайте `.holst` backup

По инструкции Holst:

1. Установить расширение Holst для импорта из Miro.
2. Открыть готовую Miro-доску.
3. Нажать `Скачать holst-бэкап этой доски`.
4. Получить файл `.holst`.

## Шаг 3. Импортируйте в Holst

В Holst:

1. Открыть рабочее пространство.
2. Нажать `Импорт`.
3. Выбрать `Из .holst backup-файла`.
4. Загрузить скачанный `.holst`.

# Рекомендуемый путь

Для скорости: используйте `Вариант 1`.

Для красивого переноса с объектами: сначала собрать в Miro, затем импортировать в Holst через `.holst`.

## Минимальный рабочий сценарий на 20 минут

1. Создать 9 фреймов в Holst.
2. Открыть `HOLST_IMPORT_Карточки_Эксперт_Сити.csv`.
3. Перенести карточки по фреймам.
4. Выделить красным фрейм `03. Квартальный фокус`.
5. Начать работу с карточки `Next actions`.
