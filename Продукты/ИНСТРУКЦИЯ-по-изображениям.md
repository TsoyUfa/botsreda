# 📸 Инструкция по изображениям для одностраничника "Аудит 360° ИИ"

## Структура папки для изображений

Создайте папку `images` в той же директории, где находится HTML-файл, и разместите следующие изображения:

```

images/
  ├── slide-1-hero.jpg
  ├── slide-2-essence.jpg

  ├── slide-6-assistant.jpg
```

## Описание изображений для каждого слайда

## Технические требования

- **Формат:** JPG или PNG
- **Размер:** минимум 1200x400px (рекомендуется 2400x800px для Retina дисплеев)
- **Вес:** до 500KB на изображение (оптимизированные)
- **Цветовое пространство:** RGB
- **Стиль:** Современный, профессиональный, в цветовой гамме сайта (фиолетово-синие оттенки)

## Где найти изображения

1. **Unsplash** (unsplash.com) — бесплатные качественные фото
2. **Pexels** (pexels.com) — бесплатные стоковые фото
3. **Freepik** (freepik.com) — иллюстрации и графика
4. **Canva** (canva.com) — создание собственных изображений
5. **Generated AI** — Midjourney, DALL-E, Stable Diffusion для создания уникальных изображений

## Рекомендации по созданию

Если используете AI для генерации, используйте такие промпты:
- "Modern business dashboard with analytics charts, professional photo"
- "AI artificial intelligence technology concept, purple blue gradient"
- "Real estate development team working together, professional office"
- "WhatsApp business messaging interface on smartphone"
- "ROI growth chart showing positive trends, financial analytics"

**Примечание:** Если изображения отсутствуют, на сайте будет отображаться плейсхолдер с текстом "📸 Место для изображения". Изображения загружаются автоматически, если они находятся в папке `images/` с указанными именами файлов.
