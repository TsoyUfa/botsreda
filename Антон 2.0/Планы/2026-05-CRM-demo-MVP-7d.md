# План MVP: CRM-демо + оффер «аудит за 7 дней»

**Статус:** в работе  
**Проект:** `real-estate-crm-html/`  
**Цель:** продаваемый демо-актив для агентств новостроек → пилоты консалтинга / CRM-диагностики  
**Фильтр:** 500k+ RUB/мес, win-win (агентство + застройщик через прозрачность)

---

## Гипотеза монетизации

| Элемент | Значение |
|---------|----------|
| **Кто платит** | РОП / собственник агентства 20–100 агентов |
| **Боль** | лиды теряются, нет следующего касания, нет причины отказа, РОП не видит воронку |
| **Оффер** | «За 7 дней: карта потерь в CRM + демо-дашборд + 3 правила SLA» |
| **Цена пилота** | 50–150k RUB (диагностика) → 300k+ (внедрение / сопровождение) |
| **Демо** | публичная ссылка на HTML CRM + 15-мин разбор |
