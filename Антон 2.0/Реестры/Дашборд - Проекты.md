# Дашборд - Проекты

Минимальные поля для проектов:
`type:: project`, `status:: active|paused|done`, `priority:: 1-3`, `deadline:: YYYY-MM-DD`, `area::`, `next::`.

## Активные
```dataview
TABLE status, priority, area, deadline, next
FROM ""
WHERE type = "project" AND status = "active"
SORT priority ASC, deadline ASC
```

## Пауза
```dataview
TABLE status, area, deadline, next
FROM ""
WHERE type = "project" AND status = "paused"
SORT deadline ASC
```

## Завершенные
```dataview
TABLE status, area, deadline
FROM ""
WHERE type = "project" AND status = "done"
SORT deadline DESC
```
