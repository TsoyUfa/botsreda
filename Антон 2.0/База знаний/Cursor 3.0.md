## 📊 КАТЕГОРИЯ 1: МАРКЕТИНГ И ПРОДАЖИ
  
### 1. 🤖 Нейрокопирайтер для постов
- **Суть применения:** генерация постов и серий под разные площадки с сохранением голоса бренда.
- **Входные данные:** кейсы, продукты, аудитория, tone of voice, ограничения платформ.
- **Реализация в Cursor:** собрать короткий бриф → сделать шаблон промпта на пост/серию → пакетно генерировать и провести редактуру.
- **Выход/результат:** контент‑план и набор готовых постов.
- **Почему это полезно:** высокая повторяемость задачи даёт быстрый эффект по времени и стоимости.
  
### 2. 📧 Email‑маркетолог (автоворонки)
- **Суть применения:** проектирование цепочек писем и триггеров для прогрева.
- **Входные данные:** сегменты аудитории, этапы воронки, боли/выгоды, триггерные события.
- **Реализация в Cursor:** описать сегменты и цели писем → задать структуру цепочки → сгенерировать тексты и проверить на соответствие бренду.
- **Выход/результат:** последовательность писем и карта триггеров.
- **Почему это полезно:** письма масштабируются и напрямую влияют на конверсию и LTV.
  
### 3. 🎯 Аналитик рекламных кампаний
- **Суть применения:** расчёт CAC, LTV, ROMI и рекомендации по оптимизации.
- **Входные данные:** отчёты рекламных кабинетов, CRM, расходы, период.
- **Реализация в Cursor:** выгрузить данные в таблицы → поручить расчёт метрик → получить выводы и гипотезы улучшений.
- **Выход/результат:** аналитический отчёт с решениями.
- **Почему это полезно:** экономит время аналитика и помогает быстро перераспределять бюджет.
  
### 4. 💬 Чат‑бот для лидогенерации
- **Суть применения:** квалификация лидов, запись на встречу, интеграция с CRM.
- **Входные данные:** вопросы квалификации, критерии лида, правила эскалации, интеграции.
- **Реализация в Cursor:** описать сценарии диалогов → создать шаблоны ответов → связать с CRM/календарём.
- **Выход/результат:** готовый сценарий чат‑бота и база ответов.
- **Почему это полезно:** снижает нагрузку на менеджеров и повышает качество лидов.
  
### 5. 📊 Анализатор отзывов
- **Суть применения:** сбор отзывов, выявление паттернов и генерация ответов.
- **Входные данные:** отзывы с площадок, категории тем, правила тональности.
- **Реализация в Cursor:** собрать отзывы → классифицировать темы и тональность → подготовить ответы по шаблону.
- **Выход/результат:** сводка инсайтов и заготовки ответов.
- **Почему это полезно:** быстро реагирует на обратную связь и улучшает репутацию.
  
### 6. 🎨 Креативщик для рекламы
- **Суть применения:** идеи креативов и тексты для баннеров, видео, Stories.
- **Входные данные:** продукт, ЦА, УТП, референсы, требования площадок.
- **Реализация в Cursor:** сделать креативный бриф → сгенерировать идеи и тексты → отранжировать варианты по гипотезам.
- **Выход/результат:** пакет креативов и текстов для теста.
- **Почему это полезно:** ускоряет тесты и повышает шанс найти сильный креатив.
  
### 7. 📞 Анализатор звонков
- **Суть применения:** анализ транскриптов по чек‑листу и рекомендации менеджеру.
- **Входные данные:** записи/транскрипты звонков, скрипт продаж, критерии качества.
- **Реализация в Cursor:** загрузить транскрипты → проверить по чек‑листу → сформировать рекомендации.
- **Выход/результат:** отчёт по качеству звонков.
- **Почему это полезно:** экономит время РОПа и повышает качество общения.
  
### 8. 🏠 Карточки объектов (недвижимость)
- **Суть применения:** продающие описания объектов и УТП под разные площадки.
- **Входные данные:** характеристики объекта, преимущества, аудитория, формат площадки.
- **Реализация в Cursor:** создать шаблон карточки → сгенерировать тексты → адаптировать под требования площадок.
- **Выход/результат:** готовые описания для публикации.
- **Почему это полезно:** больше объектов обрабатывается без потери качества.
  
### 9. 🎁 Генератор офферов
- **Суть применения:** создание офферов под разные сегменты и A/B‑варианты.
- **Входные данные:** сегменты аудитории, боли, ценность, цена, конкуренты.
- **Реализация в Cursor:** описать сегменты → сгенерировать офферы → подготовить варианты тестов.
- **Выход/результат:** набор офферов для разных каналов.
- **Почему это полезно:** быстро увеличивает конверсию за счёт точных формулировок.
  
### 10. 📱 SMM‑менеджер для Reels/TikTok
- **Суть применения:** идеи и сценарии коротких видео.
- **Входные данные:** продукт, аудитория, формат контента, цели.
- **Реализация в Cursor:** определить контент‑пиллары → сгенерировать идеи и сценарии → добавить чек‑лист съёмки.
- **Выход/результат:** пакет сценариев и план публикаций.
- **Почему это полезно:** стабилизирует производство контента без выгорания команды.
