# ИИ-пульт

Цель: быстрый и предсказуемый результат. Минимум «воды», максимум практики.

## Стандарт ответа (по умолчанию)
- Процесс: пошагово.
- Стратегия: 2–3 варианта + плюсы/минусы + рекомендация.
- Мнение: четкая позиция + аргументы.
- Анализ: разбор, прогноз, кейсы.
- Вывод: что делать дальше.

Если данных не хватает — честно сказать, что нужно, и где это взять.

## Минимальный бриф для ИИ
- Контекст: что за проект/ситуация.
- Цель: что получить на выходе.
- Ограничения: сроки, формат, стиль, ресурсы.
- Данные: факты/цифры/ссылки (если есть).
- Критерии качества: что считаем «хорошо».

## Золотые промпты (шаблоны)
1) Стратегия
```
Контекст: <описание>
Цель: <результат>
Ограничения: <сроки/ресурсы>
Сделай 2–3 стратегии с плюсами/минусами. Рекомендуй лучшую.
Дай риски и первые 5 шагов.
```

2) Пошаговый план
```
Задача: <описание>
Срок: <когда нужно>
Ресурсы: <люди/инструменты>
Сделай план по шагам, укажи зависимости, метрики и «первый шаг сегодня».
```

3) Критический разбор
```
Идея/решение: <текст>
Разбери слабые места и скрытые риски. Предложи 2 альтернативы.
```

4) Контент-брив
```
Тема: <тема>
Аудитория: <кто>
Цель: <подписка/лид/продажа>
Сделай: хук, 5 тезисов, CTA, 3 варианта заголовка.
```

5) Саммари встречи/звонка
```
Транскрипт: <вставить текст>
Дай: краткое резюме, решения, задачи, риски, следующие шаги.
```

6) Процесс/автоматизация
```
Процесс: <описание>
Цель: <метрика>
Сделай схему процесса, узкие места, автоматизацию, контроль качества.
```

7) Анализ рынка/конкурентов
```
Тема: <рынок/ниша>
Фокус: <регион/сегмент>
Сделай обзор с источниками и дай практические выводы.
```

## Быстрый выбор инструмента
- Cursor: код, автоматизация, структурирование файлов.
- ChatGPT: стратегия, копирайт, планы, шаблоны.
- Perplexity: поиск с источниками.
- NotebookLM: анализ ваших PDF/документов.
- Gamma: презентации и лендинги.

Полный список: [[Рабочая десятка]].

## Контроль качества результата
- Что можно применить сегодня?
- Где риски и слабые места?
- Какие данные нужно добавить?
- Что делать первым шагом?
