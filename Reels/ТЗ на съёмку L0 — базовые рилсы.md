# ТЗ на съёмку L0 — базовые рилсы

**Цель:** снять 3 базовых рилса за один блок (45–55 сек каждый).
**Реестр:** [[Reels/Контент-граф — способы покупки]]
**Дедлайн съёмки:** до 27.06.2026

---

## Локация и техника

| Параметр | Значение |
| :--- | :--- |
| Место | Офис Эксперт Сити (однотонный фон) |
| Свет | Качественный студийный, без теней на лице |
| Звук | Петличный микрофон |
| Кадр | Говорящая голова, средний/крупный план |
| Формат | Вертикаль 9:16, до 60 сек |
| Монтаж | Субтитры + крупный текст на экране по сценарию |

---

## Порядок съёмки (один день)

| # | ID | Сценарий | Ориентир по времени |
| :--- | :--- | :--- | :--- |
| 1 | L0-ТРАНШ | [[Reels/L0/L0-ТРАНШ — Что такое траншевая ипотека]] | ~10 мин (3 дубля) |
| 2 | L0-СУБСИД | [[Reels/L0/L0-СУБСИД — Что такое субсидированная ипотека]] | ~10 мин |
| 3 | L0-РАССР | [[Reels/L0/L0-РАССР — Что такое рассрочка]] | ~10 мин |

**Пауза между роликами:** 2–3 минуты (смена текстовых плашек на экране).

---

## Чеклист перед публикацией

- [ ] Все 3 ролика смонтированы с субтитрами
- [ ] В озвучке **нет** кодовых слов (ГРИНВИЧ, АТЛАНТИС, ДЁМА, АУДИТ)
- [ ] В озвучке **нет** конкретных ставок — только механика
- [ ] Описание под каждым рилсом скопировано из сценария
