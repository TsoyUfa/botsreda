# Карусель (серия слайдов)

## Входные данные (запроси, если нет)
- Платформа (ограничения по длине строк на слайде).
- Число слайдов (или диапазон, напр. 7–10).
- Тема и главный вывод, который должен получить читатель.
- ЦА и контекст.
- Нужен ли слайд с CTA / «следующий шаг».

## Процесс
1. Зафиксируй **одну** центральную мысль карусели.
2. Слайд 1 — обложка: короткий цепляющий заголовок + подзаголовок (если уместно).
3. Слайды 2…N-1 — по одному тезису: заголовок слайда + 1–3 короткие строки текста.
4. Последний слайд — вывод и CTA (или отдельный слайд «итог» + финальный призыв).
5. Отдельно набросай **подпись к посту** (текст под каруселью): контекст, 1–2 предложения, хэштеги по правилам площадки.

## Пример «плохо / хорошо»
- **Плохо:** слайд с 6 длинными абзацами и повтором одного и того же.
- **Хорошо:** один тезис на слайд, короткие строки, последний слайд с одним чётким действием.
