# Идеи для Reels / коротких видео

## Входные данные (запроси, если нет)
- Ниша и ЦА (что уже работает / что хочется протестировать).
- Желаемое число идей (например 10, 15, 20).
- Длительность ориентировочно (15–30 с, до 60 с, до 90 с).
- Ограничения: запрещённые темы, бренд-табу, нужна ли привязка к конкретному ЖК/продукту.

## Процесс
1. Зафиксируй 1–2 **рубрики** или угла (например «миф дня», «разбор сделки», «вопрос подписчика»), если пользователь не задал — предложи и используй дальше.
2. Для каждой идеи: хук первые 1–3 секунды, простой визуальный ряд, тезис озвучки/текста на экране, CTA в конце.
3. Избегай идей, которые невозможно снять без команды; помечай сложные пометкой «нужен монтаж».
4. Разнообразь форматы: говорящая голова, B-roll, текст на экране, тренд-аудио (если уместно).

## Формат ответа (строго по блокам)

### Рубрики / углы
- (список)

### Идея 1
- **Рабочее название:**
- **Хук (0–3 с):**
- **Визуал:**
- **Текст / озвучка (тезисно):**
- **Текст на экране (если есть):**
- **CTA:**
- **Примечания:** (длительность, «нужен монтаж», тренд и т.д.)

(повторить для идей 2…N)

### Итог
2–3 предложения: какие идеи быстрее всего снять и почему.

## Пример «плохо / хорошо»
- **Плохо:** «Снять ролик про недвижимость» — без хука и без CTA.
- **Хорошо:** хук-вопрос → один факт за 20 с → «сохрани, чтобы не потерять / напиши вопрос в комментах».
