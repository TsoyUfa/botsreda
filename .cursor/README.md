# Cursor — операционная система vault

Карта для агента: куда смотреть при разных типах задач. Obsidian-версия: `Антон 2.0/Cursor — карта навигации.md`.

## Быстрый роутинг

| Задача | Сначала прочитать | Skill / Command |
|--------|------------------|-----------------|
| Приоритет, ROI, «что делать» | `Антон 2.0/Антон 2.0 — профиль обо мне.md`, инбокс | `business-decision-router`, `/biz` |
| Разбор заметки → действие | заметка + профиль | `/knowledge`, `knowledge-execution-flow` |
| Сортировка vault | папка/файлы | `second-brain-triage` |
| CRM, воронка, SLA | `Стратегия первички/`, CRM-досье | `crm-funnel-diagnostician` |
| КП, оффер застройщику | профиль, бренд, кейсы | `real-estate-offer-architect`, `/slides` |
| Контент, пост, Reels | `anton-voice` | `social-post`, `carousel`, `reels-ideas` |
| Запуск MVP, деплой, фазы | план в `Антон 2.0/Планы/` | `cursor-workflow-coach`, `delivery-to-result-flow` |
| Фильтр «принесёт ли деньги» | цель 500k+ | `mvp-money-filter` |
| ПДн, CRM, секреты | проект с данными | `security-pd-review` |
| Метод смыслокодинга | `Практикум Смыслокодинг/Методология в Cursor — эссе.md` | `cursor-workflow-coach` |

## Слои контекста (порядок чтения)

1. **Кто думает** — `Антон 2.0/Антон 2.0 — профиль обо мне.md`, `Антон 2.0/ИИ/ИИ-пульт.md`
2. **Что за бизнес** — `Продукты/`, `Застройщики/`, `ЭКСПЕРТ СИТИ/`, `Стратегия первички/`, `Антон 2.0/Инбокс задач — май 2026.md`
3. **Как делать хорошо** — `Практикум Смыслокодинг/`, `Шаблоны контента/`, `.cursor/skills/`
4. **Текущий план** — `Антон 2.0/Планы/<актуальный>.md`
5. **Код / MVP** — `real-estate-crm-html/` и др. проектные папки

## Папки vault (модель 4+1)

| Папка | Вопрос | Примеры |
|-------|--------|---------|
| **Бизнес** | Что продаём, кому, цифры? | `Продукты/`, `Застройщики/`, `ЭКСПЕРТ СИТИ/`, CRM-досье |
| **План** | Что делаем сейчас? | `Антон 2.0/Инбокс задач — май 2026.md`, `Антон 2.0/Планы/` |
| **Мой клон** | Как думает Антон? | `Антон 2.0/`, `.cursor/rules/`, `anton-voice` |
| **Мастерство** | Как делать хорошо? | `Практикум Смыслокодинг/`, методички, промты |
