Эта библиотека содержит готовые шаблоны для быстрого запуска проектов в сфере недвижимости и обучения. Создана на основе реальных рабочих проектов и методик.

### 1. [Telegram-бот для обучения](./telegram-bot-obuchenie/README.md)
- Готовая структура бота для онлайн-обучения
- Интеграция с платежными системами

### 2. [Методичка для агентов](./metodichka-agenta/README.md)

## 🎯 Как использовать

1. **Выберите шаблон** - перейдите в папку нужного шаблона
2. **Скопируйте файлы** - создайте новую папку для проекта
3. **Настройте под себя** - измените конфигурацию, брендинг, контент
4. **Запустите** - следуйте инструкциям в README каждого шаблона
