## 🚀 Быстрый старт

### 1. Копирование шаблона

```bash
# Создайте новую папку для вашего контент-плана
mkdir "Контент-план [Название]"
cd "Контент-план [Название]"

# Скопируйте файлы шаблона
cp -r "../Библиотека шаблонов/content-plan-reels/*" .
```

### 2. Настройка контент-плана

Откройте `content-plan-template.md` и настройте:

```markdown
- Замените [Ваше имя/Бренд] на ваше название
- Укажите целевую аудиторию
- Настройте график постинга
- Адаптируйте темы под вашу нишу
```

### 3. Выбор формата

Выберите основной формат контента:

#### 🎥 Reels (30-60 секунд)
- Максимальный охват
- Виральный потенциал
- Быстрое производство

#### 📊 Карусели (5-10 слайдов)
- Глубокое погружение
- Экспертный контент
- Долгое хранение

#### 📝 Посты (текст + визуал)
- Детальные объяснения
- Документальные материалы
- Сбор комментариев

## 🎯 Метод "И что тогда?"

### Структура метода

```
1. Хук (0-3 секунды) - боль, вопрос, провокация
2. Тезис (3-15 секунд) - одна мысль, без воды
3. Развитие (15-40 секунд) - метод "И что тогда?"
4. CTA (40-45 секунд) - конкретное действие
```

### Примеры формул хуков

#### Цифра + парадокс
- "150 рублей за лида. Серьёзно?"
- "ROI 5000% на недвижимости - это реально?"

#### Разрушение мифа
- "Квартира - это не инвестиция"
- "Застройщик не ваш друг"

#### Провокация
- "Почему я отказываю 80% клиентов"
- "Ваш бизнес - это не то, что вы думаете"

#### POV (точка зрения)
- "Вот что видит агент, когда вы идёте к застройщику без него"
- "Что видит риэлтор, когда вы говорите 'нужна дешёвая квартира'"

### Пример реализации

```markdown
### "Квартира - это не инвестиция"
- **Хук:** "Если вы покупаете квартиру как инвестицию - вы теряете деньги"
- **Суть:** Квартира даёт 3-5% годовых. Инфляция съедает. Налоги, ремонт, простои. Реальная доходность около нуля.
- **Поэтому:** Я учу агентов не продавать квадратные метры, а решать задачи.
- **CTA:** Напиши "РОСТ" - пришлю расчёт реальной доходности
```

## 📋 Типы контента

### 🔥 Разрушение мифов (самые виральные)
- Цель: показать неочевидные вещи
- Формат: провокация + объяснение + решение
- Эффект: высокая вовлеченность, шеринг

### 💰 Деньги и экономика
- Цель: показать бизнес-подход
- Формат: цифры + расчеты + выводы
- Эффект: доверие, экспертный статус

### 🤖 Технологии и инновации
- Цель: показать современные подходы
- Формат: тренды + инструменты + преимущества
- Эффект: восприятие как эксперта

### 🏗️ Рынок изнутри
- Цель: эксклюзивная информация
- Формат: инсайды + данные + прогнозы
- Эффект: подписки, доверие

### 🧠 Мышление предпринимателя
- Цель: философия + методология
- Формат: принципы + системы + результаты
- Эффект: лояльная аудитория

## 🎨 Создание контента

### Шаг 1: Генерация идеи
Используйте шаблон для генерации:

```markdown
1. **Выберите тип контента:** [ ] Мифы [ ] Финансы [ ] Технологии [ ] Рынок [ ] Мышление
2. **Определите боль аудитории:** _________________________
3. **Сформулируйте хук:** _________________________________
4. **Разработайте тезис:** _______________________________
5. **Добавьте "И что тогда?":** __________________________
6. **CTA:** _____________________________________________
```

### Шаг 2: Подготовка сценария
Используйте шаблон сценария:

```markdown
## [Название Reels]

### Хук (0-3 сек)
[Текст для хука]

### Тезис (3-15 сек)
[Основная мысль]

### Развитие (15-40 сек)
[Метод "И что тогда?"]
- Поэтому...
- Но...
- Однако...

### CTA (40-45 сек)
[Конкретное действие]
```

### Шаг 3: Создание визуала
- **Формат:** говорящая голова + текст на экране
- **Стиль:** минимализм, смысл > красота
- **Текст:** крупный, читаемый, контрастный
- **Фон:** нейтральный, не отвлекающий

### Шаг 4: Монтаж
1. Нарежьте видео по сценарию
2. Добавьте текстовую дорожку
3. Наложите музыку (без авторских прав)
4. Добавьте субтитры
5. Проверьте длительность (30-60 сек)

## 📊 Аналитика и оптимизация

### Ключевые метрики
- **Просмотры:** охват аудитории
- **Вовлеченность:** лайки, комментарии, сохранения
- **Конверсия:** переходы по ссылке, подписки
- **Виральность:** шеринг, mentions

### Еженедельный отчет

```markdown

- Всего Reels: [X]

- Средняя вовлеченность: [X%]
- Лучшый контент: [Название]

### Выводы
- Что сработало: _________________________
- Что не сработало: _______________________
- Новые идеи: ____________________________
```

### Оптимизация контента
На основе аналитики:

1. **Увеличение охвата:** больше виральных тем
2. **Увеличение вовлеченности:** больше провокаций
3. **Увеличение конверсии:** более четкие CTA
4. **Улучшение качества:** анализ лучших видео
