## 🚀 Быстрый старт

### 1. Копирование шаблона

```bash
# Создайте новую папку для вашей методички
mkdir "Название методички"
cd "Название методички"

# Скопируйте файлы шаблона
cp -r "../Библиотека шаблонов/metodichka-agenta/*" .
```

### 2. Настройка методички

Откройте `metodichka-template.md` и настройте:

```markdown
- Замените [Название агентства] на ваше название
- Обновите контакты автора
- Адаптируйте скрипты под ваш стиль общения
- Добавьте специфику вашего региона/рынка
```

### 3. Кастомизация скриптов

Для каждого скрипта в папке `skripty/`:

1. Откройте файл
2. Адаптируйте под ваш стиль общения
3. Добавьте региональную специфику
4. Протестируйте на коллегах
