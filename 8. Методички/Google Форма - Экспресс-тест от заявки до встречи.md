# Google Форма: Экспресс-тест «От заявки до встречи»

Этот документ содержит структуру и инструменты для быстрого переноса экспресс-теста для опытных агентов в Google Форму.

---

## 🧭 Вариант 1. Автоматическое создание формы за 5 секунд (Рекомендуется)

Чтобы не заносить вручную все вопросы и варианты ответов, можно использовать автоматический скрипт:

1. Перейдите в браузере по ссылке: [script.new](https://script.new) (откроется пустой проект Google Apps Script под вашим аккаунтом Google).
2. Полностью удалите весь код, который там написан по умолчанию.
3. Скопируйте и вставьте в редактор JavaScript-код, приведенный ниже (в разделе «Код автоматизации»).
4. Нажмите кнопку **«Сохранить»** (иконка дискеты) или нажмите `Ctrl + S` (`Cmd + S`).
5. Нажмите кнопку **«Выполнить»** (иконка треугольника / Run).
6. Google запросит разрешения (Authorization Required). Нажмите **«Предоставить разрешения»** (Review Permissions), выберите свой аккаунт, нажмите **«Дополнительно»** (Advanced) ➔ **«Перейти к проекту...»** (Go to Untitled project (unsafe)) и нажмите **«Разрешить»** (Allow).
7. Скрипт выполнится за 2 секунды. В окне «Журнал выполнения» (Execution log) появятся ссылки на готовую форму:
   * **Published URL** — ссылка для отправки агентам (для прохождения теста).
   * **Editor URL** — ссылка для редактирования формы и просмотра ответов.
8. Ссылка на созданную форму также появится на вашем [Google Диске](https://drive.google.com). Скопируйте ссылки и вставьте их в начало этого файла для удобного доступа.

### 💻 Код автоматизации (Google Apps Script)

```javascript
function createExpressTestForm() {
  // Создаем новую форму
  var form = FormApp.create('Экспресс-тест: От заявки до встречи');
  form.setDescription('Этот тест предназначен для закрепления ключевых понятий, скриптов и логики первого контакта с клиентом по методике «Навигатор».');
  form.setIsQuiz(true); // Включаем режим теста с автопроверкой
  
  // Вопрос 1
  var q1 = form.addMultipleChoiceItem();
  q1.setTitle('1. Какая последовательность шагов от первого контакта до встречи является правильной по методике?')
    .setChoices([
      q1.createChoice('А) Звонок ➔ Отправка планировок в Telegram ➔ Показ ➔ Договор.', false),
      q1.createChoice('Б) Первое касание (Звонок/Чат) ➔ Финансовый аудит ➔ JTBD-Диагностика ➔ Создание Финмодели ➔ Первая встреча/Презентация.', true),
      q1.createChoice('В) Звонок ➔ Одобрение ипотеки ➔ Показ всех ЖК в выбранном районе.', false)
    ])
    .setPoints(1);
    
  // Вопрос 2
  var q2 = form.addMultipleChoiceItem();
  q2.setTitle('2. В чем главная опасность отправки буклетов и прайсов клиенту в Telegram сразу после первого запроса (без диагностики)?')
    .setChoices([
      q2.createChoice('А) Агент тратит слишком много личного времени на отправку файлов.', false),
      q2.createChoice('Б) Агент превращается в бесплатное «справочное бюро» для клиента, сливая ценность своей экспертизы и саму сделку.', true),
      q2.createChoice('В) Застройщик оштрафует агентство за преждевременное разглашение цен.', false)
    ])
    .setPoints(1);

  // Вопрос 3
  var q3 = form.addMultipleChoiceItem();
  q3.setTitle('3. С чего по «Железному правилу новостроек» всегда начинается подбор объекта?')
    .setChoices([
      q3.createChoice('А) С выбора района, этажа и планировок.', false),
      q3.createChoice('Б) С определения финансовой схемы (банки, лимиты, ПВ) и только в самом конце — подбор конкретных ЖК.', true),
      q3.createChoice('В) С визита в отдел продаж ключевого застройщика города.', false)
    ])
    .setPoints(1);

  // Вопрос 4
  var q4 = form.addMultipleChoiceItem();
  q4.setTitle('4. Как правильно объяснить клиенту, почему мы не начинаем подбор сразу с конкретного района (например, Зеленой Рощи)?')
    .setChoices([
      q4.createChoice('А) Сказать, что в Зеленой Роще сейчас нет хороших предложений, и предложить другой район.', false),
      q4.createChoice('Б) Объяснить, что сейчас решает финансовый инструмент (рассрочка, транши, субсидии). Ограничившись одним районом, клиент может упустить программу в соседнем ЖК, которая сэкономит ему 1.5 млн рублей на старте.', true),
      q4.createChoice('В) Сказать, что сначала нужно заключить договор на подбор недвижимости, а потом мы начнем искать.', false)
    ])
    .setPoints(1);

  // Вопрос 5
  var q5 = form.addMultipleChoiceItem();
  q5.setTitle('5. Что такое JTBD-запрос клиента при покупке квартиры?')
    .setChoices([
      q5.createChoice('А) Это техническое задание (комнатность, площадь кухни, этажность).', false),
      q5.createChoice('Б) Это истинная жизненная задача, которую решает покупка (например: безопасность семьи от выселения, личные границы для детей, надежный сейф от инфляции).', true),
      q5.createChoice('В) Это анкета банка для проверки кредитоспособности клиента.', false)
    ])
    .setPoints(1);

  // Вопрос 6
  var q6 = form.addMultipleChoiceItem();
  q6.setTitle('6. Вокруг каких двух экспертных инструментов строится закрытие звонка и продажа первой встречи/созвона?')
    .setChoices([
      q6.createChoice('А) Скидка от агентства и подарок при бронировании.', false),
      q6.createChoice('Б) Финмодель (сравнительная таблица расчетов) и Маршрутная карта (план точечных выездов по объектам-фаворитам).', true),
      q6.createChoice('В) Буклет застройщика и бесплатное такси до офиса продаж.', false)
    ])
    .setPoints(1);

  // Вопрос 7
  var q7 = form.addMultipleChoiceItem();
  q7.setTitle('7. Что такое LTV в бизнесе риелтора (Lifetime Value)?')
    .setChoices([
      q7.createChoice('А) Это соотношение суммы кредита к стоимости недвижимости при оценке залога банком.', false),
      q7.createChoice('Б) Это пожизненная ценность клиента — общая сумма комиссионных, которую один клиент приносит риелтору за всё время сотрудничества через повторные сделки и рекомендации.', true),
      q7.createChoice('В) Стоимость привлечения одного нового клиента из таргетированной рекламы.', false)
    ])
    .setPoints(1);

  // Вопрос 8
  var q8 = form.addMultipleChoiceItem();
  q8.setTitle('8. Что такое LTV в ипотечном кредитовании (Loan-to-Value)?')
    .setChoices([
      q8.createChoice('А) Отношение суммы кредита к стоимости покупаемой недвижимости (определяет размер первоначального взноса).', true),
      q8.createChoice('Б) Процентная ставка банка с учетом субсидии от застройщика.', false),
      q8.createChoice('В) Срок действия кредитного решения по заемщику.', false)
    ])
    .setPoints(1);

  // Вопрос 9
  var q9 = form.addMultipleChoiceItem();
  q9.setTitle('9. Какова главная цель совместного выезда с клиентом на строительную площадку/показ?')
    .setChoices([
      q9.createChoice('А) Послушать презентацию от менеджера застройщика.', false),
      q9.createChoice('Б) Провести тест-драйв вживую: оценить качество отделки, надежность застройщика, динамику стройки и «пощупать» будущий образ жизни ногами.', true),
      q9.createChoice('В) Выбрать планировку и узнать точную цену квартиры (которые мы и так знаем).', false)
    ])
    .setPoints(1);

  // Вопрос 10
  var q10 = form.addMultipleChoiceItem();
  q10.setTitle('10. Что должен сделать агент сразу после согласования шорт-листа с клиентом и перед показом?')
    .setChoices([
      q10.createChoice('А) Попросить клиента подписать эксклюзивный договор.', false),
      q10.createChoice('Б) Зафиксировать уникальность клиента в CRM выбранных застройщиков, чтобы закрепить условия, скидки и партнерскую комиссию.', true),
      q10.createChoice('В) Сразу ехать на стройку без предупреждения застройщика.', false)
    ])
    .setPoints(1);

  // Вопрос 11
  var q11 = form.addMultipleChoiceItem();
  q11.setTitle('11. Какая фраза при первом контакте правильно снимает барьер недоверия клиента к риелторам?')
    .setChoices([
      q11.createChoice('А) «Я подберу вам самый лучший вариант, потому что я лучший риелтор в городе».', false),
      q11.createChoice('Б) «Моя работа по подбору, расчетам и сопровождению для вас полностью бесплатна — ее оплачивает застройщик. При этом все цены, скидки и акции остаются такими же, как напрямую в отделе продаж».', true),
      q11.createChoice('В) «Я могу договориться с застройщиком о дополнительной скидке лично для вас».', false)
    ])
    .setPoints(1);

  Logger.log('Published URL: ' + form.getPublishedUrl());
  Logger.log('Editor URL: ' + form.getEditUrl());
}
```
