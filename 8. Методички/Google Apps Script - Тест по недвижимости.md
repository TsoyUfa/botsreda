# 📝 Google Apps Script: Автоматическое создание теста по недвижимости

## 🎯 Обзор скрипта

Этот скрипт автоматически создает Google Form с тестом по недвижимости из 10 ключевых вопросов. Скрипт можно запустить прямо из Google Sheets или как самостоятельный проект.

---

## 🔧 Как использовать скрипт

### **Шаг 1: Создать новый Google Apps Script проект**
1. Перейдите на [script.google.com](https://script.google.com)
2. Нажмите "Новый проект"
3. Скопируйте код ниже в редактор

### **Шаг 2: Настройка проекта**
1. Дайте проекту имя (например, "Создание теста по недвижимости")
2. Нажмите "Сохранить"

### **Шаг 3: Запуск скрипта**
1. Нажмите "Запустить" (Run)
2. При первом запуске может потребоваться авторизация
3. Разрешите доступ к Google Forms

## 📜 Полный код скрипта

```javascript
/**
 * Создает Google Form с тестом по недвижимости
 * @returns {string} URL созданной формы
 */
function createRealEstateTest() {
  // Создаем новую форму
  var form = FormApp.create('Тест по недвижимости для риелторов')
    .setDescription('Этот тест проверяет знания ключевых аспектов работы с недвижимостью. Внимательно читайте вопросы и выбирайте наиболее точные ответы.')
    .setAllowResponseEdits(true)
    .setCollectEmail(true)
    .setLimitOneResponsePerUser(true)
    // .setRequireLogin(true) // Удалите или закомментируйте эту строку на личных аккаунтах @gmail.com (требуется Google Workspace)
    .setShowProgressBar(true)
    .setShuffleQuestions(false);

  // Настраиваем прогресс-бар и настройки
  form.setProgressBar(true);
  form.setShuffleQuestions(false);

  // Добавляем вопросы
  addQuestionsToForm(form);

  // Получаем URL формы
  var formUrl = form.getPublishedUrl();
  
  // Логируем результат
  Logger.log('Форма создана: ' + formUrl);
  
  // Возвращаем URL
  return formUrl;
}

/**
 * Создает форму с ответами для проверки
```

```javascript
    .setTitle('Что для вас лично будет главным критерием того, что обучение прошло успешно?')
    .setRequired(true);

  var formUrl = form.getPublishedUrl();
  Logger.log('Анкета создана: ' + formUrl);
  return formUrl;
```

## 🚀 Как запустить скрипт

### **Способ 1: Прямой запуск**
1. Откройте [script.google.com](https://script.google.com)
2. Создайте новый проект
3. Скопируйте код выше
4. Нажмите "Запустить" (▶️)
5. Выберите функцию `createRealEstateTest`
6. Разрешите доступ (требуется только первый раз)

### **Способ 2: Через Google Sheets**
1. Создайте новую Google Таблицу
2. Меню: `Расширения` → `Apps Script`
3. Скопируйте код
4. Запустите функцию

### **Способ 3: Добавление в меню Google Sheets**
