# Четыре Reels по методу «И что тогда?»

Готовый пакет к съёмке: блоки, сплошной текст для суфлёра, тайминг, проверка на запретную связку «и», локализация и A/B для CTA.

---

## Чеклист перед записью (анти-«и»)

- [ ] Между смысловыми шагами — только **Поэтому** / **Но** / **Однако**, не перечисление через «и».
- [ ] После каждого блока внутри головы: «**и что тогда?**» — следующая фраза отвечает на это.
- [ ] На **Но** / **Однако** — короткая пауза (удержание внимания).
- [ ] Прогон текста вслух один раз; вычеркнуть «и ещё», «и просто», «и тогда», если это лишь дополнение факта, а не новый поворот.

## Reel 1 — Новостройки / ипотека (ограниченный бюджет + капитал)

**Хук на экран:** «Центр без раздувания первого взноса»

| Роль | Текст |
|------|--------|
| Тезис | Клиент хотел квартиру ближе к центру, но бюджета на большой первый взнос не было. |
| *И что тогда?* | |
| Поэтому | Мы не стали тащить его во вторичку с размыванием первого взноса под высокую ставку. |
| *И что тогда?* | |
| Но | Нашли новостройку с траншевой ипотекой: символический платёж до ключей. |
| *И что тогда?* | |
| Поэтому | Пока дом достраивается, деньги лежат на вкладе — контракт уже фиксирует цену. |

**Сплошной текст (суфлёр):**  
Клиент хотел квартиру ближе к центру, но бюджета на большой первый взнос не было. Поэтому мы не стали тащить его во вторичку с размыванием первого взноса под высокую ставку. Но нашли новостройку с траншевой ипотекой — символический платёж до ключей. Поэтому пока дом достраивается, деньги лежат на вкладе, а контракт уже фиксирует цену.

**Ориентир по таймингу (30–40 с):**

| Сек | Кадр / реплика |
|-----|----------------|
| 0:00–0:03 | Хук на экран + первая реплика тезиса |
| 0:03–0:12 | Поэтому (вторичка / ставка) |
| 0:12–0:22 | Но (траншевая / платёж до ключей) |

## Reel 2 — Новостройки / выбор застройщика (цена vs полная стоимость)

**Хук на экран:** «Самая низкая цена за метр — не всегда самая низкая сделка»

| Роль | Текст |
|------|--------|
| Тезис | Покупатель заклинился на самой дешёвой ₽/м² по объявлению. |
| *И что тогда?* | |
| Поэтому | Я разложил ему не ценник, а полную стоимость входа: сроки, ипотека, скрытые платежи, риски по этапам. |
| *И что тогда?* | |
| Однако | «Дешевле» по объявлению ломалось на условиях банка и наценках на отделку. |
| *И что тогда?* | |
| Поэтому | Взяли чуть дороже по объявлению — дешевле по факту выхода на ключи и по нервам. |

**Сплошной текст:**  
Покупатель заклинился на самой дешёвой цене за метр по объявлению. Поэтому я разложил ему не ценник, а полную стоимость входа: сроки, ипотека, скрытые платежи, риски по этапам. Однако «дешевле» по объявлению ломалось на условиях банка и наценках на отделку. Поэтому мы взяли чуть дороже по объявлению — но дешевле по факту выхода на ключи и по нервам.

**Тайминг:**

| Сек | Блок |
|-----|------|
| 0:00–0:05 | Хук + тезис |

## Reel 4 — Консалтинг / процессы перед автоматизацией

| Роль | Текст |
|------|--------|
| Тезис | Руководитель хотел «ещё одну CRM», хотя сделки застревали между чатами и таблицами. |
| *И что тогда?* | |
| Поэтому | Сначала описали путь клиента от лида до ДДУ — нашли три узла, где терялись деньги и время. |
| *И что тогда?* | |
| Однако | Без правил для партнёров и SLA по ответам любая новая система дала бы тот же хаос. |
| *И что тогда?* | |
| Поэтому | Автоматизацию включили после этого — окупается за счёт скорости, не «красивых отчётов». |

**Сплошной текст:**  
Руководитель хотел ещё одну CRM, хотя сделки всё равно застревали между чатами и таблицами. Поэтому мы сначала описали путь клиента от лида до ДДУ и нашли три узла, где терялись деньги и время. Однако без правил для партнёров и SLA по ответам любая новая система давала бы тот же хаос. Поэтому автоматизацию включили только после этого — и она начала окупаться за счёт скорости, а не красивых отчётов.

## Локализация под вашу практику

Подставьте свои данные **до записи** — одно-два значения на ролик достаточно.

| Reel | Поле | Ваш ввод |
|------|------|----------|
| 1 | Город / локация «центр» | |
| 1 | Пример: ставка / название программы (если можно озвучить) | |
| 2 | Город или типичный ЖК / формат отделки в регионе | |
| 2 | Одна цифра-коридор (например платёж или переплата) — по желанию | |
| 3 | Доля времени или лидов (если заменяете «почти половину») | |
| 3 | Отрасль (если не только застройщик) | |
| 4 | CRM / чаты (если называете вслух) | |

**Пример вставки в Reel 1 (не читать списком — одной фразой):**  
Тезис остаётся общим; в блоке «Но» можно сказать: «В ___ нашли программу, где до ключей платёж символический» — без цепочки «и ещё вот это».

## A/B тест CTA: Reel 1 и Reel 3

Публикуйте в одинаковое время недели, одну обложку/один хук, меняйте только финальную фразу. Фиксируйте **7 дней**: охват, сохранения, ответы в директ с ключевым словом.

### Reel 1

| Вариант | Текст CTA |
|---------|-----------|
| **A (директ-лид)** | Напишите город и примерный бюджет — скину 2–3 рабочих схемы. |
| **B (сохранения)** | Сохраните: перед показом квартир так проверяем вход в сделку без раздувания первого взноса. |

**Гипотеза:** A даёт больше директа; B — больше сохранений и охват за счёт «полезности».

### Reel 3

| Вариант | Текст CTA |
|---------|-----------|
| **A (ключевое слово)** | Поток лидов и просадка конверсии — напишите «разбор», пришлю чек-лист. |
| **B (калибровка)** | Если у вас больше 300 лидов в месяц и узкое горло в первой линии — напишите «узкое место», скину схему прогрева. |

**Гипотеза:** A шире по охвату ответов; B отсекает шум и повышает качество директа.

### Как сравнить

| Метрика | Где смотреть |
|---------|----------------|
| Сохранения | Инсайты Reels |
| Ответы в директ | Вручную; метка чата «R1-A», «R1-B», «R3-A», «R3-B» |
| Конверсия в созвон | Ваша CRM / календарь |

После теста оставьте победивший CTA в основном шаблоне и заархивируйте второй для повторного теста через 4–6 недель.

## Съёмка и монтаж (напоминание)

- Кадр: говорящая голова или ключевая фраза на экране по шагам.
- Музыка: без текста, громкость под голос.
- Физическая съёмка и выкладка — у вас на стороне; этот файл закрывает подготовку сценария и постановку эксперимента по CTA.
