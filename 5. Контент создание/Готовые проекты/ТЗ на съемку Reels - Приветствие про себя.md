# ТЕХНИЧЕСКОЕ ЗАДАНИЕ НА СЪЕМКУ REELS

**Формат:** «Знакомство / Кто я и зачем вам это слушать» (приветствие, про себя)
**Хронометраж:** 35-45 секунд
**Настроение:** Уверенное, спокойное, инженерно-четкое. Без мотивационного пафоса.

## 1. Визуальная часть (Мизансцена)

- **Локация:** Минималистичный фон. Темная стена или ТВ-панель с приглушенным светом. Без визуального шума.
- **Внешний вид:** Темная однотонная футболка/рубашка базового цвета. Casual-минимализм. Без принтов и пестрых деталей.
- **Ракурс:** Средний план (по пояс). Говорим прямо в камеру, как с живым человеком один на один.
- **Свет:** Мягкий рассеянный. Лицо освещено равномерно, фон темнее на 2-3 ступени. Премиальный темный тон по дизайн-коду РОСТ.
- **Движение в кадре:** Спокойное, без суеты. Легкие жесты руками для акцентов. Между блоками - микропаузы (0.5 сек), которые вырезаются при монтаже.

## 2. Монтаж и Типографика (Постпродакшн)

- **Шрифт:** Крупный, жирный, без засечек (Montserrat / Space Grotesk / Plus Jakarta Sans).
- **Цвет текста:** Основной - белый. Акцентные слова и цифры - бирюзовый (`#00f2fe`) или мятный (`#39ff14`) по дизайн-коду РОСТ.
- **Фон видеоряра:** Deep Noir (`#030611`), темный премиальный тон.
- **Анимация текста:** Появление слов синхронно с речью. Короткие фразы - по одному-два слова на экране, не перегружать.
- **Склейки:** Jump cut между смысловыми блоками. Убрать все паузы, вдохи, запинки. Темп плотный, но не суетливый.
- **Музыка:** Тихий атмосферный бит на фоне. Без вокала. Громкость - 10-15% от голоса.

## 4. Чек-лист съемки

- [ ] Камера на уровне глаз, горизонталь ровная
- [ ] Горизонтальная или вертикальная стабилизация (без тряски)
- [ ] Чистый звук с петлички или внешнего микрофона
- [ ] 2-3 дубля каждого блока для выбора лучшего
- [ ] Темная однотонная одежда без принтов
- [ ] Фон без лишних предметов и бликов
- [ ] Паузы между блоками для удобного монтажа

## 5. Правила текста (3П)

- Не врать
- Не приукрашивать
- Не подлизываться
- Без англицизмов (где есть русские аналоги)
- Без мотивационного пафоса
- Каждое утверждение подкреплено личным опытом или цифрой

## Ключевые факты для импровизации (если захотите сказать своими словами)

- Образование: техникум по специальности АСУ (Автоматизированные системы управления), Институт информационных технологий
- Первый масштаб: перенос бумажного справочника 2ГИС в цифру
- Корпоративный опыт: оцифровка процессов для сетей (Пышка, Папа Гриль, Даско, Лидер)
- Кризис: 3-4 года назад ушел в свой бизнес, столкнулся с хаосом и выгоранием
- Инструменты: Obsidian (Второй мозг) + ИИ-клон через Телеграм-бот
- Результат: 5 проектов параллельно вместо 1-2
- Методология: РОСТ (Регламенты, Оцифровка, Свобода, Технологии)
