# ТЕХНИЧЕСКОЕ ЗАДАНИЕ НА СЪЕМКУ REELS

**Формат:** «Разрушение мифов / Ожидание vs Реальность» (ИИ для агентов по недвижимости)  
**Хронометраж:** 30–45 секунд  
**Настроение:** Уверенное, экспертное, успокаивающее, структурное.  

## 1. Визуальная часть (Мизансцена)

* **Локация:** Минималистичный фон. ТВ-панель со светлым фоном (голубой, серый) или чистая маркерная доска/стена.
* **Реквизит:** Квадратные карточки (стикеры) двух цветов:
  * **Цвет 1 (розовый/красный):** Для мифов/ожиданий.
  * **Цвет 2 (зеленый):** Для реальности/цифр.
* **Внешний вид:** Однотонная темная футболка или casual-рубашка, брюки. Без принтов.
* **Ракурс:** Средний план. Доска с карточками справа, спикер слева.

## 2. Логика взаимодействия в кадре

* **Подготовка:** Красные карточки с мифами уже на доске. Зеленые карточки с реальностью — в руках.
* **Действие:** На каждый тезис приклеиваем зеленую карточку слева от красной.
* **Взгляд:** В момент приклеивания — на доску. При озвучивании цифры и опровержения — уверенный взгляд в камеру.

## 3. Монтаж и Типографика (Постпродакшн)

* **Шрифт:** Крупный, жирный, без засечек (Montserrat / Roboto / Apple-стиль).
* **Цвет текста:** Белый. Акцентные слова (цифры, «а не») — желтые или зеленые.
* **Анимация текста:** Появление слов синхронно с речью, текст не перекрывает доску.
* **Склейки:** Динамичный jump cut между тезисами без длинных пауз.
