**Собрано:** 24 мая 2026  
**Режим:** оригиналы перенесены из прежних мест (не копии)

1. **Снять Reels сегодня** → `01` + `09/Личный бренд/Идеи на реализацию`
2. **Запланировать неделю** → `04/Контент-план на 30 дней.md`
3. **Вести Telegram** → `03/Идеи для ТГ канала.md` + `billb/`
4. **Проверить ToV** → `06/Брендбук` + `09/Личный бренд/Tone of Voice.md`
5. **ИИ для съёмки** → `08/Обучение - ИИ для недвижимости/`
