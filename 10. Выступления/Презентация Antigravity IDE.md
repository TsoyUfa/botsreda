## 🧬 ЧАСТЬ 1. ФИЛОСОФИЯ ЭКРАНА И ДИЗАЙН-КОД

Эта презентация построена на принципе полного слияния спикера с цифровым пространством. Экран не дублирует слова — он реагирует на физические жесты, визуализируя невидимые процессы автоматизации.

### 📐 Правила трансляции смыслов:
* **Правило 7 слов:** Ни один слайд не содержит более 7 слов.
* **Спикер как контроллер:** Каждое ключевое изменение на экране инициируется жестом, касанием или шагом спикера.
* **Глубокий черный фон (#1a1a1a):** Убирает свечение границ экрана, делая фигуру спикера трехмерной.

```
Схема расположения зон на экране 3х5м:
+-------------------------------------------------+
|                    ЗОНА В                       |
|  +-------------------------------------------+  |
|  |                 ЗОНА Б                    |  |
|  |  +-------------------------------------+  |  |
|  |  |              ЗОНА А                 |  |  |
|  |  |         (Центр внимания)            |  |  |
|  |  |                                     |  |  |
|  |  +-------------------------------------+  |  |
|  |                                           |  |
|  +-------------------------------------------+  |
|                                                 |
+-------------------------------------------------+
* Зона А (Центр): Ключевые слова, фокусные цифры, объекты касания.
* Зона Б (Средняя): Схемы, таймлайны, интерактивные панели.
* Зона В (Периферия): Неоновые частицы, фоновые связи, сетка HUD.
```

## 🎪 ЧАСТЬ 3. ПОШАГОВЫЙ СЦЕНАРИЙ И ХОРЕОГРАФИЯ

### Слайд 1. Интро: «Код мертв»
* **Смысловой фокус:** Снять романтику с программирования, захватить внимание предпринимателей жестким тезисом.

#### 📊 Визуальное воплощение на экране:
Абсолютно черный экран. В центре (Зона А) горит одно массивное слово **«КОД»**, выполненное в виде цианового неонового стекла. Вокруг него медленно пульсирует тусклая сетка.

```
+-------------------------------------------------+
|                                                 |
|                                                 |
|                    [ КОД ]                      |
|                (неоновое стекло)                |
|                                                 |
|                                                 |
+-------------------------------------------------+
```

#### 🕴️ Хореография спикера:
Антон стоит в Позиции 2 (0.5м от экрана), лицом к слову «КОД», спиной к залу. Руки опущены. В зале тишина.

#### ⚡ Интерактивный триггер и анимация:
Антон медленно поднимает руку и касается ладонью слова «КОД».
* **В момент контакта:** Из-под ладони с низкочастотным звуком удара разбегаются оранжевые трещины.
* **Через 0.5 секунды:** Слово со звоном разбивается на миллионы циановых пикселей, которые улетают вниз экрана. Экран гаснет.
* **Действие:** Антон плавно поворачивается к залу в Позицию 1 (1.5м от экрана, вполоборота) и начинает говорить.

#### 🗣️ Сценарий ToV:
> «Написание кода вручную умерло. Нанимать штат разработчиков, чтобы два месяца собирать простой лендинг или автоматизировать воронку — это слив денег. Сегодня мы управляем не кодом, а смыслом. Я объясню, как это работает».

* **Вариант короче:** «Синтаксис больше не важен. Разработка в старом понимании мертва. Мы перешли к управлению намерениями».

### Слайд 2. Архитектура: «4 папки SoloOps»
* **Смысловой фокус:** Показать структуру Второго мозга, без которого никакой ИИ-агент работать не сможет.

#### 📊 Визуальное воплощение на экране:
На экране четыре серых неактивных блока (квадранты РОСТ) с тонкими границами.

```
+------------------------+------------------------+
|                        |                        |
|      [ БИЗНЕС ]        |        [ ПЛАН ]        |
|      (серый блок)      |      (серый блок)      |
|                        |                        |
+------------------------+------------------------+
|                        |                        |
|     [ МОЙ КЛОН ]       |     [ МАСТЕРСТВО ]     |
|      (серый блок)      |      (серый блок)      |
|                        |                        |
+------------------------+------------------------+
```

#### 🕴️ Хореография спикера:
Антон стоит по центру в Позиции 1, лицом к залу. При упоминании каждого блока он делает шаг к экрану в Позицию 2 и указывает на него рукой.

#### ⚡ Интерактивный триггер и анимация:
Антон по очереди тапает по блокам:
1. `БИЗНЕС` ➔ Загорается циановым. Внутри раскрывается: *«Цены, КП, тарифы»*.
2. `ПЛАН` ➔ Загорается зеленым. Внутри: *«Алгоритмы, KPI, запуски»*.
3. `МОЙ КЛОН` ➔ Загорается оранжевым. Внутри: *«Голос автора, ценности»*.
4. `МАСТЕРСТВО` ➔ Загорается белым. Внутри: *«Скрипты, регламенты»*.
* **Финал:** Все 4 блока соединяются неоновыми линиями в единый центр — Внутреннее ядро ИИ-ассистента.

#### 🗣️ Сценарий ToV:
> «ИИ не может работать без контекста. Если вы дадите ему пустой лист — получите банальные ответы. Мы делим бизнес на 4 папки: Бизнес, План, Мой Клон и Мастерство. Это ваш Второй мозг в Obsidian. И только после этого мы подключаем к нему агента».

* **Вариант короче:** «4 папки в Obsidian — это фундамент. Контекст вашего бизнеса, оцифрованный один раз. Без этого ИИ бесполезен».

* **Вариант короче:** «Хватит копипастить. Агент берет задачу, сам пишет код, запускает тесты и показывает вам готовый результат».

### Слайд 6. Результат: «Конвейер SoloOps»
* **Смысловой фокус:** Показать финальную цепочку сборки продукта силами одного человека.

#### 📊 Визуальное воплощение на экране:
Горизонтальный таймлайн в Зоне Б: `[Идея] ➔ [Код] ➔ [Тест] ➔ [Деплой]`.

```
+-------------------------------------------------+
|                                                 |
|  [Идея] ----> [Код] ----> [Тест] ----> [Деплой] |
|                                                 |
|          (таймлайн с неоновыми узлами)          |
|                                                 |
+-------------------------------------------------+
```

#### 🕴️ Хореография спикера:
Антон медленно идет вдоль экрана слева направо, последовательно касаясь каждого узла таймлайна.

#### ⚡ Интерактивный триггер и анимация:
При касании каждого узла:
1. `Идея` ➔ Загорается оранжевым (Obsidian Draft).
2. `Код` ➔ Загорается циановым (Antigravity Agentic Write).
3. `Тест` ➔ Загорается зеленым (Terminal Tests OK).
4. `Деплой` ➔ Вспыхивает золотым светом (VPS Live). Во все стороны разлетаются золотые частицы.

#### 🗣️ Сценарий ToV:
> «Вот цепочка SoloOps. Вы фиксируете идею в Obsidian. Агент пишет код. Сам прогоняет тесты в терминале на ошибки. И разворачивает проект на сервере. Весь цикл занимает меньше часа и выполняется силами одного оператора. Без лишних встреч и согласований».

* **Вариант короче:** «Один человек управляет всей цепочкой: от идеи до деплоя. Агент кодит, тестирует и выкатывает на сервер за один цикл».

## 🛠️ ЧАСТЬ 4. ТЕХНИЧЕСКИЙ СЕТАП И ЗАПУСК ПРЕЗЕНТАЦИИ

Чтобы презентация работала интерактивно и плавно на большом экране, выберите один из режимов настройки:

### ⏱️ Сценарий MVP (Запуск за 1 час)
* **Инструменты:** Apple Keynote или MS PowerPoint.
* **Переход:** **Magic Move (Волшебное перемещение)** для всех слайдов.
* **Синхронизация:** Спикер держит в руке мини-кликер. Жест рукой делается за 0.2 секунды до нажатия кнопки на кликере. Для зала создается полная иллюзия бесконтактного управления.

### 🚀 Сценарий PRO (Настоящий интерактив)
* **Инструменты:** Интерактивное веб-приложение (HTML5 Canvas / CSS3 Transitions), запущенное локально.
* **Управление:** iPad в руках спикера (дублирует экран в режиме контроллера) или инфракрасная сенсорная рамка (IR Touch Overlay), смонтированная поверх экрана. Каждое физическое касание экрана спикером мгновенно запускает JS-события анимации.
