## 🧬 ЧАСТЬ 1. 핵심-ИНСАЙТЫ КОНСУЛЬТАЦИИ («ЖИР»)

Перед созданием презентации зафиксируем фундаментальные смыслы, вытащенные из консультации с психологом. Это концептуальный клей всей презентации:

1. **Концепт «Второй мозг без первого не бывает»**
   * ИИ сам по себе — пустая коробка. Он копирует банальные шаблоны из интернета, если в него не загружен контекст конкретного человека. 
   * Задача эксперта — не писать промпты, а оцифровать и передать ИИ свое **мастерство и контекст**. Если ты сам не разбираешься в теме, ИИ заведет тебя в ахинею.

2. **Быстрый пайплайн выгрузки знаний (NotebookLM + Obsidian)**
   * Вместо месяцев мучений: эксперт записывает на диктофон сырой поток мыслей (монолог без структуры на 20-30 минут) -> загружает аудио в NotebookLM -> ИИ транскрибирует, вырезает воду, разделяет Q&A и вытаскивает чистую суть -> материал переносится во Второй мозг (Obsidian) по 4 папкам.

3. **Специфика ИИ-ассистентов для помогающих практиков (на примере психолога)**
   * Вместо генерации стандартных постов, ИИ настраивается под глубокие профессиональные задачи:
     * **Анамнез и рефлексия:** Анализ аудиозаписей прошлых сессий, выявление паттернов клиента.
     * **Подготовка к супервизии:** Автоматическая сборка клинического случая, подготовка тезисов для супервизора по методологическим стандартам.

4. **Воронка «Продажа через сложность»**
   * Полная открытость: спикер дает клиентам огромную, детальную методичку (20+ страниц), описывающую всю систему под ключ. 
   * **Психологический триггер:** Клиент читает методичку, убеждается в феноменальной экспертности спикера, охуевает от сложности самостоятельной реализации («ебал я рот это делать сам») и платит за готовое внедрение опыта.

5. **Правило холодного контента**
   * Аккаунт в соцсетях должен быть оформлен так, чтобы абсолютно незнакомый человек с первого взгляда понял: *кто ты, что ты, куда вести и зачем*. Нельзя вести блог так, будто все вокруг уже «свои» и знают контекст.

### Слайд 1. Интро: «ИИ тупит»
* **Смысловой фокус:** Разбить иллюзию «волшебной кнопки» ИИ, показать главную боль — банальность выдачи ChatGPT.

#### 📊 Визуальное воплощение на экране:
Абсолютно черный экран. В центре (Зона А) горит одно ярко-оранжевое неоновое слово: **«ИИ ТУПИТ»**. Под ним мелким шрифтом (70% белого): *«Шаблонный бред вместо решений»*.

```
+-------------------------------------------------+
|                                                 |
|                                                 |
|                   [ ИИ ТУПИТ ]                  |
|            (неоновый оранжевый акцент)          |
|         *Шаблонный бред вместо решений*         |
|                                                 |
|                                                 |
+-------------------------------------------------+
```

#### 🕴️ Хореография спикера:
Антон стоит в Позиции 2 (лицом к экрану, 0.5м). Рука поднята в жесте касания слова «ИИ ТУПИТ». 

#### ⚡ Интерактивный триггер и анимация:
Антон делает легкое нажатие пальцем на слово.
* **В момент касания:** Из точки контакта расходится оранжевая волна искажения (glitch-эффект). Текст осыпается пикселями.
* **Действие:** Антон разворачивается в Позицию 1 (3/4 к залу, 1.5м от экрана).

#### 🗣️ Сценарий ToV:
> «Вы заходите в ChatGPT, просите написать пост или план продаж, а он выдает вам водянистую банальщину. ИИ не глупый. Просто у него нет контекста. Вы не дали ему первый мозг».
