# Инструменты

## Основные

- Obsidian - второй мозг, заметки, контекст, досье, проекты.
- Cursor - AI-интерфейс для работы с заметками, файлами, кодом и стратегией.
- ChatGPT / Claude / Gemini - генерация, анализ, сравнение подходов.
- Telegram - коммуникации, контент, боты, воронки.

## Автоматизация

- CRM - сделки, лиды, статусы, контроль воронки.
- API - интеграции с CRM, банками, застройщиками, таблицами.
- Google Sheets / Яндекс Таблицы - быстрые расчёты и MVP.
- Боты - заявки, уведомления, квалификация, сбор данных.

## Бизнес-артефакты

- Презентации.
- Коммерческие предложения.
- Лендинги.
- Чек-листы.
- Скрипты продаж.
- Диагностические формы.
- Карты процессов.

## Правило выбора инструмента

Инструмент выбирается после ответа на вопрос: какой бизнес-результат он должен дать?

# Инструменты

## Текущий стек

- Obsidian - второй мозг, база заметок, стратегий, досье и проектов.
- Cursor - рабочий интерфейс для анализа, упаковки, автоматизации и работы с файлами.
- Telegram - канал коммуникации, контента и продаж.
- Презентации - упаковка продуктов, обучений и коммерческих предложений.
- Таблицы - финансы, контакты, трекеры, расчёты.

## Практичный AI/automation стек

- ChatGPT / Claude / Gemini - генерация, анализ, стратегия, тексты.
- CRM - воронка, сделки, лиды, контроль.
- API - интеграции застройщиков, банков, CRM и сервисов.
- Google Sheets / Яндекс Таблицы - быстрые MVP, расчёты, базы.
- Telegram-боты - коммуникации, лидогенерация, напоминания, сбор данных.

## Как выбирать инструмент

- Если нужно быстро проверить гипотезу - таблица, презентация, ручной процесс.
- Если процесс повторился 3+ раза - можно думать об автоматизации.
- Если есть много ручных передач данных - API/CRM.
- Если нужно сохранить и переиспользовать знания - Obsidian.
- Если нужно упаковать оффер - презентация или landing-структура.
