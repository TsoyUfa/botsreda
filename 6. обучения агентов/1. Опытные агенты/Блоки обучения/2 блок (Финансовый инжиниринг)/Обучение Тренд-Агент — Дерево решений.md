## ⚡ ЭКСПРЕСС-ДИАГНОСТИКА: 5 вопросов за 2 минуты

```
ШАГ 1. Есть дети до 7 лет / беременна?
        ДА  →  Семейная ипотека 3.5–6% (приоритет №0)
        НЕТ ↓

ШАГ 2. Первоначальный взнос ≥ 20%?
        НЕТ →  Рассрочка «Накопи на ПВ» (Архстрой 3.4%) 
                или КапиталЦентр (10%)
        ДА  ↓

ШАГ 3. Дом сдаётся через 1+ год?
        ДА  →  Есть депозит / вторичка под продажу?
               ДА  →  ТРАНШЕВАЯ ИПОТЕКА
               НЕТ ↓

ШАГ 4. Покупает для жизни на 5+ лет?
        ДА  →  СУБСИДИРОВАННАЯ СТАВКА (12–14%)
        НЕТ ↓

ШАГ 5. Есть крупный ПВ (30–50%+)?
        ДА  →  РАССРОЧКА от застройщика (+ переход в ипотеку)
        НЕТ →  Разбираем индивидуально
```

| Профиль клиента | Главная боль | Инструмент | Ключевой аргумент |
|:---|:---|:---|:---|
| **Арендатор-ожидатель** (снимает, боится двойного платежа) | 35к аренда + 38к ипотека = 73к | **Транш 100к** | Платёж 600–1 800 ₽/мес до сдачи. Экономия 900к за 2 года |
| **Вторичник-переселенец** (есть квартира, вторичка стоит) | Продать с дисконтом -1.5 млн | **Рассрочка 10/90** | 18 мес спокойной продажи. Риск = 0 |
| **Депозитный рантье** (деньги на вкладе, «подожду») | «Заработаю и куплю за наличку» | **Транш + арбитраж** | Вклад работает + цена зафиксирована. Транш = 1 800 ₽/мес |
| **Молодая семья с декретом** (доход упадёт) | Не вытянем 50к/мес на одну зарплату | **Транш** → **Семейная ипотека** | Платёж 1 800 ₽/мес до рождения. После ребёнка → 6% |
| **Консерватор на 10+ лет** (покупает для жизни) | «Не хочу переплачивать» | **Субсидированная ставка** | Удорожание окупается за 12–22 мес. Экономия 4 млн за 10 лет |
| **Ипотечный маневрист** (нет льгот, ставка 22% пугает) | «Подожду снижения ставок» | **Рассрочка → ипотека** | Фиксируем цену сегодня + ипотека по низкой ставке завтра |
| **Бизнесмен в обороте** (деньги приносят 30%+) | «Вынимать деньги из бизнеса — терять прибыль» | **Рассрочка 30/70** | Бизнес работает. Рассрочка окупается из оборотной прибыли |
| **Инвестор** (100% наличных, «не люблю долги») | «Зачем мне ипотека?» | **Транш + депозит** | 4.8 млн на депозите 18% → +1.7 млн за 2 года. Транш = 24 ₽ |

## 🚨 КОГДА КАКОЙ ИНСТРУМЕНТ НЕ ПОДХОДИТ

### Транш — НЕ для:
- Клиентов без чёткого источника денег через 2 года
  *(нет депозита, нет вторички, нет гарантированного роста дохода)*
- Тех, кто «просто надеется» рефинансироваться — без конкретного плана

### Субсидия — НЕ для:
- Тех, кто планирует продать квартиру через 1–2 года
  *(удорожание не успеет окупиться)*
- Тех, кто хочет досрочно закрыть за год
  *(дешевле взять без удорожания под рыночную и быстро погасить)*

### Рассрочка — НЕ для:
- Клиентов без понятного источника остатка суммы к сдаче дома

## 💬 ФРАЗЫ ДЛЯ ДИАГНОСТИКИ (аудит за 2 минуты)

| Вопрос агента | Что выясняем |
|:---|:---|
| «Сколько у вас готово сейчас — наличными?» | Размер ПВ и ликвидность |
| «Есть деньги на вкладе? Когда заканчивается?» | Источник для погашения транша |
| «Сдаёте или снимаете сейчас жильё?» | Двойная нагрузка — нужен транш |
| «Есть квартира, которую хотите продать?» | Вторичка → рассрочка-мост |
| «Откуда у вас появятся деньги через 2 года?» | Стресс-тест плана клиента |
| «Планируете детей в ближайшие 1–2 года?» | Декрет + семейная ипотека после |
| «Берёте для себя или как инвестицию?» | Горизонт владения → выбор инструмента |

## 🔑 ЗОЛОТЫЕ ФРАЗЫ НАВИГАТОРА

**Вместо «цена квартиры»** → *«стоимость обслуживания долга»*

**Вместо «переплата»** → *«инвестиция в дешёвые деньги»*

**Вместо «берите сейчас»** → *«давайте посчитаем два сценария»*

**Вместо «удорожание»** → *«окупается за 12 месяцев»*

**Вместо «подождите, банки откроют лимиты»** → *«зафиксируем бронь бесплатно, подаём в ДОМ.РФ»*
