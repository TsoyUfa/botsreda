# 📋 Шпаргалка: Способы покупки новостроек

*Одностраничник для агента — держи перед глазами при разговоре с клиентом*

---

## ⚡ Экспресс-подбор: 5 вопросов → ответ

```
1. Есть дети? → ДА → Семейная ипотека 3,5-6% (приоритет №0)
                → НЕТ ↓
2. ПВ ≥ 20%?  → НЕТ → Рассрочка: Накопи на ПВ (3,4%) / КапиталЦентр (10%)
                → ДА ↓
3. Дом строится 1+ год? → ДА → Есть депозит? → ДА → Траншевая ипотека
                                                → НЕТ ↓
4. Нужен низкий платёж?  → На 1 год → Совкомбанк (⚠️ без МК!)
                          → На весь срок → Субсидия 12-14%
5. Скоро сдача?          → ПВ ≥ 50% → Рассрочка 50/50
                          → ПВ < 50% → Субсидия или рыночная
```

## 🚩 5 красных флагов (проговори клиенту!)

1. **Обрыв платежа** — субсидия 0,1% → 19-22% с 13-го мес.
2. **Обрыв при транше** — траншевая: рост в 15-50 раз при вводе
3. **Скрытое удорожание** — «бесплатная» рассрочка = +5-10к/м²
4. **Маткапитал запрещён** — Совкомбанк «Сниженный платёж»
5. **Ключи до 100% оплаты** — почти все рассрочки: нет

## 💡 Комбо-схемы (продвинутый уровень)

1. **Рассрочка «Накопи на ПВ»** → **Траншевая** (Архстрой: копим 20,1%, потом транш)
2. **Рассрочка 25/25/50** → **Ипотека** на остаток при снижении ключевой (КПД)
3. **Траншевая + депозит** → **Досрочка** при финальном транше (классический арбитраж)
