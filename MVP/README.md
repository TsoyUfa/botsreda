# 🚀 Набор ИИ-прототипов (MVP) в твоем Обсидиане

В этой папке находятся 5 готовых к работе решений на базе искусственного интеллекта (Gemini API) для автоматизации твоей работы, обучения брокеров и аналитики.

---

## 📦 Быстрый старт

### 1. Установка зависимостей
Убедись, что на твоем Mac установлен Python 3. Запусти терминал в этой папке и выполни команду:
```bash
pip install -r requirements.txt
```

### 2. Настройка ключей
Создай файл `.env` на основе шаблона `.env.example`:
```bash
cp .env.example .env
```
Открой `.env` и вставь туда свой `GEMINI_API_KEY` (и `TELEGRAM_BOT_TOKEN`, если планируешь использовать бот для команды).

---

## 🛠️ Описание решений и инструкции по запуску

### 1. ИИ-Тренажер возражений брокера (`1_agent_trainer_simulator`)
Интерактивный симулятор, в котором ты можешь тренировать отработку возражений («субсидии — это развод», «дорого» и др.). ИИ играет роль клиента, а в конце выдает оценку и обратную связь по твоему «Чек-листу первого касания».
*   **Запуск:**
    ```bash
    python 1_agent_trainer_simulator/app.py
    ```
    Открой в браузере: `http://127.0.0.1:8000`

### 2. Оцифровщик планерок и встреч (`2_meeting_summarizer`)
Записывай планерку или созвон с брокером на аудио, закидывай файл в папку `/Обсидиан/inbox/` и запускай скрипт. Он транскрибирует запись через Gemini API (поддерживается прямая загрузка аудио) и сделает структурированный Markdown-отчет с задачами, сроками и инсайтами.
*   **Запуск:**
    ```bash
    python 2_meeting_summarizer/summarizer.py
    ```

### 3. Семантический линкер Второго Мозга (`3_semantic_linker`)
Скрипт анализирует все твои заметки в Obsidian, находит смысловые сходства и создает отчет «Рекомендованные связи» с предложением вставить ссылки `[[заметка]]` на похожие статьи.
*   **Запуск:**
    ```bash
    python 3_semantic_linker/linker.py
    ```

### 4. ИИ-Ассистент брокеров (`4_team_rag_assistant`)
Telegram-бот для твоей команды, который отвечает на любые вопросы менеджеров строго по материалам из папки `/6. обучения агентов/` твоим голосом (`anton-voice`).
*   **Запуск:**
    ```bash
    python 4_team_rag_assistant/bot.py
    ```

### 5. Анализатор воронки CRM (`5_crm_funnel_analytics`)
Положи выгруженный из Битрикс24 отчет сделок (в формате CSV) в папку `5_crm_funnel_analytics` под именем `deals.csv` и запусти аудит. ИИ проанализирует сделки, найдет узкие места, провалы в конверсии брокеров и выдаст качественный отчет.
*   **Запуск:**
    ```bash
    python 5_crm_funnel_analytics/audit.py
    ```
