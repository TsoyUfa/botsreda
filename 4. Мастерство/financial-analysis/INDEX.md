# Financial Analysis: деньги, ROI, юнит-экономика

## Задача раздела

Методы для оценки идей, продуктов и сделок в рублях: доход, маржа, окупаемость, риск, масштаб.

## Когда читать

- Перед выбором продукта.
- Перед расчётом оффера.
- Перед внедрением автоматизации.
- Когда нужно понять, приближает ли идея к 500к+ рублей в месяц.
