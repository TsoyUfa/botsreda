# Raw

Сюда можно складывать сырые материалы, из которых потом делаются файлы мастерства.

## Что можно класть

- транскрипты выступлений;
- расшифровки созвонов без персональных данных;
- заметки по книгам;
- статьи и конспекты;
- черновики методологий.

## Что нельзя класть

- телефоны, email, паспортные данные клиентов;
- токены, пароли, ключи;
- полные книги, если нет прав;
- большие неразобранные материалы без цели.

## Как перерабатывать

Промпт для Cursor:

```text
Прочитай этот raw-файл и выдели только рабочие методы.
Формат каждого метода: что это / когда применять / шаги / результат / какие данные добавить.
Не делай пересказ. Не добавляй воду. Если данных не хватает, поставь [добавить инфу: ...].
Создай или обнови файл в нужной папке 4. Мастерство/<область>/.
```

# Raw: сырые материалы

## Зачем эта папка

Сюда можно складывать исходники, из которых потом делается концентрат для `4. Мастерство`: транскрипты, черновики, разборы, заметки по авторам, фрагменты выступлений.

## Правило

Сырые материалы не являются готовым мастерством. Из них нужно выделить методы и перенести в тематические файлы:

```text
raw/транскрипт.md -> область/автор.md
```

## Что можно добавлять

- Транскрипты выступлений экспертов.
- Расшифровки созвонов.
- Фрагменты книг или статей для анализа.
- Черновики своих наблюдений.

## Что нельзя добавлять

- Пароли, токены, ключи.
- Персональные данные клиентов.
- Конфиденциальные условия сделок без обезличивания.
- Огромные материалы без задачи на выжимку.

## Шаблон обработки

```text
Источник:
Автор / эксперт:
Область:
Что извлечь:
Какие 1-3 метода выделены:
Куда перенести:
```

## Что добавить

- [добавить инфу] первые 1-3 транскрипта или заметки, из которых нужно сделать выжимку.
- [добавить инфу] список экспертов, чьи методы Антон хочет встроить в систему.
