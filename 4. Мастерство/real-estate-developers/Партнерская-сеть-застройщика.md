# Партнерская сеть застройщика

> Линза: застройщику нужна не «база агентов», а управляемый канал продаж с понятной мотивацией, коммуникацией, обучением и контролем заявок.

## Когда применять

- Упаковка продукта «партнерская сеть под ключ».
- Диагностика отношений застройщик - агентства.
- Создание КП для девелопера.
- Разработка MVP сервиса или консалтинга.

## Метод 1. Карта партнерского канала

Шаги:

1. Кто партнеры: агентства, частные агенты, ипотечные брокеры, банки, смежные сервисы.
2. Что они хотят: быстрые ответы, прозрачную комиссию, обучение, актуальные остатки, доверие.
3. Что хочет застройщик: заявки, сделки, контроль, прогноз, качество клиентов.
4. Где трение: нет коммуникации, долго отвечают, непонятные правила, слабое обучение, CRM не отражает реальность.
5. Какие 3 узких места можно улучшить за 7 дней.

Выход:

- карта текущего партнерского канала;
- список потерь;
- 3 быстрых улучшения;
- предложение пилота.

## Метод 2. Win-win оффер для партнеров

Суть: агент включается, когда видит свою выгоду и простоту действий.

Проверка оффера:

- Понятно, сколько можно заработать?
- Понятно, как быстро получить ответ?
- Есть материалы для продажи?
- Есть обучение по объекту?
- Есть прозрачный статус клиента?
- Есть человек/бот/канал для поддержки?

## Метод 3. MVP партнерской сети

Минимальная проверка без большого продукта:

1. Выбрать 1 ЖК / 1 объект.
2. Найти 10-20 теплых агентов.
3. Дать им понятный пакет: презентация, условия, чат поддержки, скрипт, FAQ.
4. Отслеживать заявки в простой таблице или CRM.
5. Через 7-14 дней измерить: активность, заявки, встречи, сделки, возражения.
