# ПЕРЕСТРОЙКА БИЗНЕСА - Стратегия на 90 дней

**Дата создания:** 2025-01-27  
**Для:** Агентство недвижимости (60 агентов, 3 руководителя)  
**Автор:** Антон Цой

---

## КОНТЕКСТ И ЦЕЛИ

### Текущая ситуация

**Агентство:**
- 60 агентов недвижимости
- 3 руководителя
- Работает в районе с большим количеством новостроек
- Сильная команда с ресурсами

**Проблемы:**
- Низкое качество лидов
- Высокая стоимость привлечения клиентов
- Неэффективные процессы работы
- Нет системы работы с тёплой базой

**Метрики:**
- Средний чек новостройки: 350 000 ₽
- Средний чек вторички: 170 000 ₽

### Цели перестройки

**Через 90 дней:**
- Улучшение качества лидов на 50-70%
- Снижение стоимости лида на 50-70%
- Рост количества сделок на 50-100%
- Рост конверсии лид → сделка на 30-50%
- Автоматизация 60-70% рутинных процессов
- Рост производительности агентов на 30-50%

## ФАЗА 1: ДИАГНОСТИКА И СТРАТЕГИЯ (Дни 1-14)

### Неделя 1: Аудит текущей ситуации

#### День 1-2: Сбор метрик

**Задачи:**
1. Собрать данные по привлечению клиентов:
   - Количество лидов в месяц по каждому каналу
   - Стоимость лида (CPL) по каждому каналу
   - Конверсия лид → встреча по каналам
   - Конверсия встреча → сделка по каналам
   - Время от лида до первого контакта

2. Собрать данные по продажам:
   - Количество сделок в месяц (новостройки/вторичка)
   - Средний чек по типам сделок
   - Производительность агентов (сделок/агент/мес)
   - Конверсия воронки по этапам

3. Собрать финансовые данные:
   - Затраты на маркетинг и рекламу (по каналам)
   - Затраты на персонал
   - Выручка и маржа
   - ROI по каналам привлечения

**Инструменты:**
- CRM система (если есть)
- Финансовые отчёты
- Интервью с руководителями
- Опросы агентов

**Результат:** Таблица "Текущие метрики агентства"

#### День 3-4: Анализ процессов

**Задачи:**
1. Проанализировать процесс привлечения лидов:
   - Какие каналы используются
   - Как работает каждый канал
   - Где теряются лиды
   - Какие каналы наиболее эффективны

2. Проанализировать процесс обработки лидов:
   - Как распределяются лиды между агентами
   - Время ответа на лид
   - Квалификация лидов
   - Где теряются лиды в процессе обработки

3. Проанализировать процесс работы с клиентами:
   - Как агенты работают с клиентами
   - Какие этапы воронки
   - Где теряются клиенты
   - Какие возражения чаще всего встречаются

4. Проанализировать процессы агентства:
   - Как создаются описания объектов
   - Как ведётся работа с документами
   - Как планируются встречи
   - Какие рутинные процессы можно автоматизировать

**Методы:**
- Наблюдение за работой агентов
- Интервью с агентами и руководителями
- Анализ CRM данных
- Тайные покупатели

**Результат:** Документ "Анализ процессов агентства"

#### День 5-7: Оценка команды

**Задачи:**
1. ABC-сегментация агентов:
   - Категория A (топ-20%): высокие продажи, стабильные результаты
   - Категория B (середнячки-60%): средние продажи, потенциал роста
   - Категория C (слабые-20%): низкие продажи, требуют внимания

2. Оценка навыков и компетенций:
   - Знания о новостройках
   - Навыки продаж
   - Работа с возражениями
   - Использование технологий

3. Оценка мотивации и вовлечённости:
   - Уровень мотивации
   - Удовлетворённость работой
   - Готовность к изменениям
   - Потенциал развития

**Методы:**
- Анализ результатов работы
- Опросы и интервью
- Оценка компетенций
- Тестирование навыков

**Результат:** Документ "Оценка команды" с ABC-сегментацией

#### Результат недели 1

**Отчёт "Текущее состояние агентства" (20-30 страниц):**
- Раздел 1: Текущие метрики (таблицы, графики)
- Раздел 2: Анализ процессов (описание, проблемы, возможности)
- Раздел 3: Оценка команды (сегментация, навыки, мотивация)
- Раздел 4: Выводы и рекомендации

### Неделя 2: Разработка стратегии

**Задачи:**
1. Установить целевые метрики привлечения:
   - Целевая конверсия лид → встреча: > 30%

   - Целевая доля тёплых лидов: 40-50%

2. Установить целевые метрики продаж:
   - Целевая производительность агента: +30-50%
   - Целевая конверсия лид → сделка: +30-50%
   - Целевое количество сделок: +50-100%

3. Установить целевые метрики эффективности:
   - Целевая экономия времени: 15-20 часов/неделю на агента
   - Целевая автоматизация процессов: 60-70%
   - Целевой рост маржи: +20-30%

**Результат:** Таблица "Целевые метрики на 90 дней"

#### День 10-11: Разработка стратегии привлечения

2. Оптимизировать существующие каналы:
   - Анализ эффективности каждого канала
   - Перераспределение бюджета
   - Улучшение качества лидов

3. Разработать систему работы с тёплой базой:

   - Отслеживание конверсии

#### День 12-13: Планирование внедрения

**Задачи:**

**Задачи:**

## КЛЮЧЕВЫЕ МЕТРИКИ УСПЕХА

### Метрики привлечения клиентов

**До перестройки:**
- Стоимость лида: 3 000-10 000 руб
- Доля тёплых лидов: 10-20%
- Конверсия лид → встреча: 15-20%

**После перестройки (цель):**
- Стоимость лида: 150-200 руб (-50-70%)
- Доля тёплых лидов: 40-50% (+200-300%)
- Конверсия лид → встреча: > 30% (+50-100%)

### Метрики продаж

**До перестройки:**
- Конверсия лид → сделка: 5-10%
- Производительность агента: 2-3 сделки/мес

**После перестройки (цель):**
- Конверсия лид → сделка: 15-20% (+50-100%)
- Производительность агента: 3-5 сделок/мес (+50-100%)

### Метрики эффективности

**До перестройки:**
- Время на рутину: 20+ часов/неделю
- Автоматизация процессов: 10-20%

**После перестройки (цель):**
- Время на рутину: 5-10 часов/неделю (-50-75%)
- Автоматизация процессов: 60-70% (+300-600%)

### Финансовые метрики

**До перестройки:**
- Выручка: базовая
- Маржа: базовая
- ROI каналов: низкий

**После перестройки (цель):**
