# ФАЗА 2: ПРОДУКТ 2 - TELEGRAM-КАНАЛ ДЛЯ ПРИВЛЕЧЕНИЯ ПОКУПАТЕЛЕЙ

**Срок:** Месяцы 3-4 (Март - Апрель 2026)  
**Статус:** Запланировано  
**Приоритет:** Высокий

---

## ЦЕЛЬ ФАЗЫ

Создать Telegram-канал для привлечения покупателей новостроек с вирусной механикой и системой работы с базой.

**Ключевые результаты:**
- 1000-3000 "забронированных" клиентов с промокодами
- Работающая система привлечения покупателей
- Первые сделки из базы
- Стоимость привлечения клиента: 150-200 руб (в 20-50 раз дешевле таргета)

---

## МЕСЯЦ 3: СОЗДАНИЕ КАНАЛА И МЕХАНИКИ

### Неделя 1: Стратегия и контент-план

**Задачи:**
- [ ] Определить целевую аудиторию канала
  - Демография (возраст, доход, семейное положение)
  - География (Уфа и регион)
  - Мотивация покупки (первая квартира, расширение, инвестиция)
  - Каналы привлечения аудитории
  
- [ ] Разработать контент-стратегию
  - Типы контента (новости, кейсы, советы, эксклюзивы)
  - Тон общения (профессиональный, дружелюбный)
  - Частота публикаций (1-3 поста в неделю)
  - Уникальное торговое предложение
  
- [ ] Создать контент-план на 3 месяца
  - Темы постов на каждый день
  - Календарь публикаций
  - Сезонные акции и события
  - Интеграция с вирусной механикой
  
- [ ] Подготовить шаблоны постов
  - Шаблоны для разных типов контента
  - Примеры оформления
  - Рекомендации по текстам
  - Библиотека изображений

- [ ] Название и бренд-обещание канала
  - Варианты названия (выбрать 1): «Новостройки Уфы | Пульс», «УфаЖК: старты и цены», «Ключи Уфы», «Уфа.Квартиры.Новости», «Дом в Уфе», «Уфа: Новостройки 2026», «ЖК Уфа: навигатор», «Новая Уфа | квартиры», «Уфа.Недвижимость.Факты», «Новостройки Уфы: разборы»
  - Критерии отбора: коротко, понятно, геопривязка к Уфе, легко искать в Telegram, доступен username, без обещаний «самых низких цен»
  - Бренд-обещание (1 фраза): «Помогаем выбрать квартиру в новостройках Уфы на основе фактов и сравнения объектов»
  - Формула описания: «Новости, старты продаж, цены, льготы и разборы ЖК Уфы. Без слухов, с источниками»
  - Оформление: аватар с лаконичной иконкой дома/ключа + упоминание Уфы, закреп с правилами канала и ссылкой на бота

- [ ] Позиционирование канала
  - Роль: эксперт-навигация по рынку новостроек Уфы
  - Для кого: покупатели первой квартиры, семьи на расширение, инвесторы с бюджетом среднего сегмента
  - Ключевые выгоды: быстрее узнавать старты продаж и изменения цен, понимать районы и инфраструктуру, сравнивать ЖК «по полкам»
  - Доверие: только проверенные источники, ссылка на первоисточник, пометка партнерских постов
  - Тональность: спокойный, конкретный, без хайпа и «желтых» заголовков
  - Ограничения: не публикуем слухи, не обещаем скидки без подтверждения

- [ ] Продвижение канала (дистрибуция)
  - Карта размещений: список локальных Telegram-каналов Уфы по тематикам (районы, городские новости, семьи, работа, авто, ипотека)
  - Партнерские публикации: застройщики, ипотечные брокеры, агентства недвижимости, дизайнеры интерьеров
  - Telegram Ads: гео Уфа + интересы «недвижимость/ипотека», тест 5-8 креативов и 3 оффера
  - Офлайн-точки: QR-коды в офисах продаж, на мероприятиях, в ипотечных центрах
  - Кросс-постинг: сайт/лендинг агентства, VK, email/мессенджер база с ссылкой на канал
  - План запуска: 2 недели теста (небольшие бюджеты) → масштабирование рабочих площадок

- [ ] Привлечение подписчиков и воронка входа
  - Лид-магниты: «Гайд по районам Уфы 2026», «Чек-лист выбора ЖК», «Топ-5 новостроек по бюджету», «Карта инфраструктуры»
  - Вход через бота: подписка на канал → короткий квиз → выдача материала и промокода
  - Реферальная механика: бонус за приглашение друга + участие в розыгрыше
  - Прозрачный CTA: «Подпишитесь и получите подборку новостроек Уфы под ваш бюджет»
  - Онбординг: закреп с правилами, навигацией и ссылкой на подборку

- [ ] Контент-архитектура и рубрикатор
  - Рубрики: старты продаж, изменения цен, вводы и сроки, ход строительства, инфраструктура районов, «разбор ЖК», сравнения 2-3 ЖК, ипотека и льготы
  - Форматы: короткие новости, карточки с фактами, разборы, опросы, еженедельный дайджест
  - Частота: базово 3 поста в неделю (новость + разбор + полезный материал), плюс срочные анонсы по факту
  - Шаблон поста: заголовок → факт → польза/вывод → CTA (подборка/бот/вопрос)

- [ ] Метрики и тесты
  - KPI: прирост подписчиков/нед, стоимость подписчика, ER, CTR по ссылкам, конверсия в бота и заявки
  - Аналитика: Telegram-статистика + UTM-метки + единая таблица лидов
  - A/B-тесты: заголовки, офферы, визуальные превью, время публикации
  - Ритм проверки: 1 раз в 2 недели с фиксацией решений

**Результат:** Полная контент-стратегия и план на 3 месяца

### Неделя 3: Вирусная механика

**Задачи:**
- [ ] Разработать задания для участников
  - **Базовое задание:** Аватарка + био + подписка (стоимость ~150 руб/чел)
  - **Премиальное задание:** Stories + подписка (стоимость ~200 руб/чел)
  - **VIP-задание:** Stories + аватарка + друг (стоимость ~250 руб/чел)
  - Описание каждого задания
  
- [ ] Настроить систему выдачи промокодов
  - Автоматическая выдача после выполнения задания
  - Разные промокоды для разных заданий
  - Проверка выполнения заданий
  - Система модерации
  
- [ ] Создать систему отслеживания выполнения заданий
  - Проверка аватарки
  - Проверка био
  - Проверка подписки на канал
  - Проверка stories (если возможно)
  
- [ ] Подготовить награды
  - Промокоды на скидку (10К, 15К, 30К руб)
  - Старсы в Telegram
  - Кофе/подарки (опционально)
  - Главный приз (розыгрыш)

**Результат:** Полностью настроенная вирусная механика
