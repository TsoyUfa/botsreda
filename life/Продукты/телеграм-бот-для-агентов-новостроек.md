# Телеграм-бот для агентов новостроек

> **Дата создания:** 08.06.2026
> **Статус:** идея → в разработку
> **Дедлайн:** среда 11.06.2026 (презентация Центрального Парка)
> **Цель:** К среде должна быть рабочая версия с базовой информацией

---

## Концепция

Телеграм-бот-помощник для агентов недвижимости. Даёт информацию по новостройкам, учит продавать и допродаёт курсы Антона.

---

## Функционал бота

### Уровень 1. Справочник агента (базовый)
- Первичные цены по новостройкам
- Просчёты (ипотека, рассрочки, траншевые)
- Преимущества каждого ЖК
- Как продавать конкретный ЖК - подход Антона, его видение

### Уровень 2. Допродажа обучения (монетизация)
- Агент делает запрос → получает ответ → далее прилетает предложение
- «Хочешь узнать больше? Напоминаю, что такое продажный строй»
- Докидывание мыслей: в какую сторону мыслить при работе с клиентом
- Продвижение курсов Антона

### Уровень 3. Проактивный бот (удержание)
- Бот сам пишет пользователю через какое-то время
- «А какие у тебя есть вторичные объекты? Могу подобрать вариант по вторичке»
- «Переориентация клиента с вторички на новостройку - это тоже источник лидов. Ты об этом не думал?»
- Напоминания, подсказки, новые ЖК

---

## Как это работает на презентации

1. Антон выступает 10-15 минут
2. Показывает QR-код → агенты попадают в бота
3. Бот уже работает: агент запрашивает ЖК, получает цены, преимущества, подход к продаже
4. В боте есть зацепка на курсы
