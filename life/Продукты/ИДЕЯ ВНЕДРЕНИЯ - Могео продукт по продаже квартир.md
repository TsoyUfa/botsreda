# ИДЕЯ ВНЕДРЕНИЯ - Мой продукт по продаже квартир

## 📝 Суть идеи

Предложение по старту работы с новой бизнес-моделью для продукта по продаже квартир.

---

## 💡 Ключевая концепция

### Модель работы:

1. **Стартовая цена от платформы**
   - Начинаем с цены, которую устанавливает платформа (могео)
   - Это базовая цена, от которой работаем

2. **Оплата размещения на Авито**
   - Стоимость размещения объявления на Авито оплачивает **продавец**
   - Платформа не несет расходы на рекламу

3. **Льготная комиссия при продаже по цене платформы**
   - Если квартира продается **по цене, которую предложила платформа**
   - Стоимость услуг платформы **снижается до 50%** от стандартной комиссии

---

## 🎯 Преимущества модели

### Для платформы (могео):

✅ **Снижение рисков**
- Не тратим свои деньги на рекламу (Авито оплачивает продавец)
- Минимальные расходы на старте

✅ **Выравнивание интересов**
- Продавец заинтересован в продаже по цене платформы (получает скидку 50%)
- Создается мотивация работать с нашей оценкой

✅ **Контроль ценообразования**
- Платформа устанавливает оптимальную цену
- Продавец заинтересован не завышать цену (иначе нет скидки)

✅ **Снижение барьера входа**
- Продавец платит только за размещение (небольшие расходы)
- Основная комиссия зависит от результата

---

### Для продавца:

✅ **Прозрачность**
- Понятная модель: платишь за размещение, получаешь скидку при продаже по нашей цене
- Контроль расходов на старте

✅ **Стимул работать честно**
- Заинтересован продать по цене платформы (получает 50% скидку на услуги)
- Выгоднее не торговаться и продать быстрее

✅ **Минимальные риски**
- Платит только за размещение на Авито (небольшая сумма)
- Остальное — при успешной продаже

---

## 💰 Финансовая модель

### Вариант 1: Продажа по цене платформы

**Расходы продавца:**
- Размещение на Авито: ~2,000-5,000 ₽ (в зависимости от формата)
- Комиссия платформы: **50% от стандартной** (например, если стандартная 3%, то 1.5%)

**Пример:**
- Квартира: 5,000,000 ₽
- Стандартная комиссия 3% = 150,000 ₽
- Комиссия по льготной цене (50%) = **75,000 ₽**
- Размещение на Авито = 3,000 ₽
- **Итого расходы продавца: 78,000 ₽**

---

### Вариант 2: Продажа по другой цене (выше или ниже)

**Расходы продавца:**
- Размещение на Авито: ~2,000-5,000 ₽
- Комиссия платформы: **100% (стандартная ставка)**

**Пример:**
- Квартира: 5,000,000 ₽
- Комиссия 3% = **150,000 ₽**
- Размещение на Авито = 3,000 ₽
- **Итого расходы продавца: 153,000 ₽**

**Разница:** Продавец теряет 75,000 ₽ при продаже по другой цене

## 📊 Сравнение с альтернативами

### Традиционная модель (риелтор):

| Параметр | Традиционная модель | Могео модель (по цене платформы) | Могео модель (другая цена) |
|----------|---------------------|----------------------------------|----------------------------|
| Комиссия | 3-5% (150-250К на 5 млн) | **1.5% (75К)** | 3% (150К) |
| Расходы на рекламу | Включены в комиссию | **Продавец платит сам (3К)** | Продавец платит сам (3К) |
| **Итого** | **150-250К** | **78К** | **153К** |
| **Экономия** | - | **72-172К** | **0-97К** |

## 🎯 Сценарии применения

### Сценарий 1: Продавец согласен с ценой платформы

**Процесс:**
1. Платформа оценивает квартиру: **5,000,000 ₽**
2. Продавец соглашается с ценой
3. Размещение на Авито (продавец платит)
4. Продажа по цене **5,000,000 ₽**
5. Комиссия: **1.5% = 75,000 ₽** (скидка 50%)

**Результат:** Выгодно для обеих сторон

---

### Сценарий 2: Продавец хочет продать дороже

**Процесс:**
1. Платформа оценивает: **5,000,000 ₽**
2. Продавец выставляет: **5,300,000 ₽**
3. Размещение на Авито (продавец платит)
4. Продажа по цене **5,300,000 ₽**
5. Комиссия: **3% = 159,000 ₽** (без скидки)

**Результат:** Продавец получил больше, но заплатил полную комиссию

---

### Сценарий 3: Квартира продается дешевле

**Процесс:**
1. Платформа оценивает: **5,000,000 ₽**
2. Продавец выставляет: **5,000,000 ₽** (согласен)
3. Размещение на Авито (продавец платит)
4. Продажа по цене **4,800,000 ₽** (торг)
5. Комиссия: **3% = 144,000 ₽** (без скидки, т.к. не по цене платформы)

**Результат:** Скидка не применяется, т.к. продажа не по установленной цене

## 🔄 Процесс работы

### Шаг 1: Оценка

- Платформа (могео) оценивает квартиру
- Устанавливает оптимальную цену продажи
- Согласование с продавцом

### Шаг 2: Размещение

- Продавец оплачивает размещение на Авито
- Платформа размещает объявление
- Начинается продвижение

### Шаг 3: Продажа

- Если продажа **по цене платформы** → комиссия **50%**
- Если продажа **по другой цене** → комиссия **100%**

### Шаг 4: Расчет

- Расчет комиссии после сделки
- Выплата средств

## 💡 Дополнительные возможности

### Вариант А: Гибкая скидка

Вместо фиксированных 50%, можно сделать прогрессивную шкалу:

- Продажа точно по цене платформы: **50% скидка**
- Продажа в пределах ±2%: **25% скидка**
- Продажа вне этого диапазона: **без скидки**

---

### Вариант Б: Гарантия продажи

Если квартира не продается за X месяцев:
- Возврат части оплаты за размещение
- Или бесплатное продление размещения

---

### Вариант В: Пакетная модель

Продавец оплачивает:
- Базовый пакет: размещение + услуги платформы
- При продаже по цене платформы: возврат части базового пакета
