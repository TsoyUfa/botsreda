# 🚀 Быстрый запуск iOS приложения

## Шаг 1: Откройте Xcode

1. Убедитесь, что у вас установлен **Xcode 15.0+**
2. Откройте Xcode

## Шаг 2: Создайте новый проект

1. В Xcode выберите: **File → New → Project**
2. Выберите **iOS → App**
3. Нажмите **Next**
4. Заполните форму:
   - **Product Name:** `AnonymousRealEstate`
   - **Team:** Выберите вашу команду (или "None" для тестирования)
   - **Organization Identifier:** `com.antontsoy` (или ваш)
   - **Interface:** `SwiftUI` ⚠️ Важно!
   - **Language:** `Swift`
   - **Storage:** `None`
5. Выберите место для сохранения проекта
6. Нажмите **Create**

## Шаг 3: Добавьте файлы в проект

### Вариант А: Через Finder (проще)

1. Откройте папку с файлами приложения в Finder
2. Перетащите все папки (`App`, `Models`, `Views`, `Services`, `ViewModels`, `Utils`) в Xcode
3. В диалоге выберите:
   - ✅ **Copy items if needed**
   - ✅ **Create groups** (не folder references!)
   - ✅ Target: **AnonymousRealEstate**
4. Нажмите **Finish**

### Вариант Б: Через Xcode

1. В Xcode нажмите правой кнопкой на папку проекта
2. Выберите **Add Files to "AnonymousRealEstate"...**
3. Выберите все папки с файлами
4. Убедитесь, что выбран Target: **AnonymousRealEstate**
5. Нажмите **Add**

## Шаг 4: Замените стандартные файлы

1. Удалите стандартный `ContentView.swift` (если он есть)
2. Убедитесь, что в проекте есть файл `AnonymousRealEstateApp.swift` из папки `App/`

## Шаг 5: Настройте токен Telegram (опционально)

1. Откройте файл `Utils/Constants.swift`
2. Найдите строку:
   ```swift
   static let telegramBotToken = "YOUR_TELEGRAM_BOT_TOKEN_HERE"
   ```
3. Замените на ваш токен (или оставьте как есть для тестирования с мок-данными)

> 💡 **Для тестирования токен не обязателен** - приложение работает с мок-данными

## Шаг 6: Запустите приложение

1. В верхней панели Xcode выберите симулятор (например, **iPhone 15 Pro**)
2. Нажмите **⌘ + R** (или кнопку ▶️ Play)
3. Дождитесь компиляции и запуска

## Шаг 7: Тестирование

1. Приложение откроется на экране авторизации
2. Введите:
   - **Telegram ID:** `123456789` (любое число)
   - **Имя:** `Тестовый Пользователь` (любое имя)
3. Нажмите **"Войти"**
4. Вы увидите главный экран приложения!

## ✅ Готово!

Приложение должно работать с тестовыми данными. Вы можете:
- Просматривать ветки диалогов
- Открывать чаты
- Смотреть профиль
- Тестировать все функции

## 🔧 Если что-то не работает

### Ошибка компиляции

1. **Очистите проект:** `Product → Clean Build Folder` (⌘ + Shift + K)
2. **Пересоберите:** `Product → Build` (⌘ + B)
3. Проверьте, что все файлы добавлены в Target

### Приложение не запускается

1. Проверьте, что выбран симулятор (не "Any iOS Device")
2. Убедитесь, что в `AnonymousRealEstateApp.swift` есть `@main`
3. Проверьте настройки Signing & Capabilities

### Файлы не найдены

1. Убедитесь, что все папки добавлены в проект
2. Проверьте, что файлы добавлены в Target `AnonymousRealEstate`
3. В Project Navigator должны быть видны все файлы

## 📱 Запуск на реальном устройстве

1. Подключите iPhone/iPad через USB
2. В Xcode выберите ваше устройство вместо симулятора
3. В настройках проекта (`Signing & Capabilities`):
   - Выберите вашу Team
   - Xcode автоматически создаст профиль
4. Нажмите **⌘ + R**
5. На устройстве разрешите установку (если нужно)
