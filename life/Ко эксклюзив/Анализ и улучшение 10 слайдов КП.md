## Слайд 7. Управленческий ритм проекта


**Заголовок:**
Управленческий ритм: что и когда получает СУ-10

**Подзаголовок:**
Точки контроля выстроены так, чтобы вы могли принять решение о продлении пилота на основании фактов, а не ожиданий.

**Ритм отчётности:**

| Точка                             | Периодичность      | Формат                 | Содержание                                                   |
| --------------------------------- | ------------------ | ---------------------- | ------------------------------------------------------------ |
| Оперативка (СУ-10 ↔ Эксперт-Сити) | Каждый понедельник | 30 минут               | Воронка, отклонения, решения недели, ответственные           |
| Отчёт                             | Каждую пятницу     | 1 страница             | План / факт / 4 контрольные точки / решения, нужные от СУ-10 |
| Решение                           | **28–29 мая**      | Стратегическая встреча | Итоги мая, прогноз июня, условия продления                   |

**Критерии продления на июнь:**
- Выполнение плана 5 сделок в мае → автоматическое продление на июнь по той же модели
- 3–4 сделки в мае → продление с корректировкой плана и условий
- < 3 сделок в мае → выход без обязательств для обеих сторон

**От СУ-10 для пилота:** один ответственный с правом принимать оперативные решения. Это снимает 80% риска «согласования тормозят канал».

## Слайд 10. Условия и решение

**Подзаголовок:**
Нулевые дополнительные затраты застройщика. Оплата только за факт сделки.

**Финансовая модель:**

| Текущая практика СУ-10                 | Предложение Эксперт-Сити                 |
| -------------------------------------- | ---------------------------------------- |
| Скидка покупателю — 10%                | Скидка покупателю — **5%**               |
|                                        | Комиссия Эксперт-Сити — **5%**           |
| **Итог для застройщика: –10% от цены** | **Итог для застройщика: –10% от цены**   |

**Эффект для СУ-10:** ноль изменений уровня цен, ноль дополнительных затрат, плюс работающий партнёрский канал.

**Условия апробации:**
- Период: **май 2026** (1 месяц)
- KPI: **5 закрытых сделок**
- При выполнении — продление на июнь и июль на той же модели
- При невыполнении — выход обеих сторон без обязательств
- Оплата — **только по факту проведённой сделки** через банк-партнёр

**Решение, которое нужно от вас сегодня:**
1. Подтверждение запуска пилота мая на условиях 5%+5%
2. Назначение ответственного со стороны СУ-10
3. Старт подготовки 5–6 мая, запуск офлайн-блока 13–15 мая
