# Операционный план — СУ-10: 90 дней

**Назначение:** рабочая таблица внедрения системы эксклюзивных продаж.  
**Логика:** 30 дней пилота → 30 дней усиления → 30 дней закрепления и решения о масштабировании.  
**Главные KPI:** брони, активность партнёров, SLA реакции и решения вопросов.

## Роли

| Роль                      | Краткое обозначение  | Зона ответственности                               |
| ------------------------- | -------------------- | -------------------------------------------------- |
| Куратор проекта           | Куратор              | Общая координация, решения, связь с СУ-10          |
| Партнёрский менеджер      | Партнёрский менеджер | Агентства, агрегаторы, ABC-сегментация, касания    |
| Отдел заботы              | Забота               | Единая точка входа, вопросы, SLA, эскалации        |
| Тренер/эксперт по объекту | Тренер               | Обучения, продуктовая аргументация, материалы      |
| Аналитик                  | Аналитик             | Данные, отчёты, воронка, контроль метрик           |
| Ответственный СУ-10       | СУ-10                | Согласования, данные, условия, продуктовые решения |

## Минимальная форма отчёта

| Метрика                | Что фиксируем                                        | Частота                |
| ---------------------- | ---------------------------------------------------- | ---------------------- |
| Active-партнёры      | Количество партнёров с реальными касаниями за неделю | Еженедельно            |
| Обучения               | Дата, тема, участники, вопросы                       | После каждого обучения |
| Вопросы партнёров      | Тема, источник, статус, срок решения                 | Ежедневно              |
| SLA                    | Время первой реакции и полного решения               | Ежедневно              |
| Брони                  | Партнёр, объект, статус, следующий шаг               | Ежедневно              |
| Конфликты уникальности | Стороны, причина, решение                            | По факту               |
| Решения СУ-10          | Что нужно согласовать, кто отвечает, срок            | Еженедельно            |

## Правила корректировки плана

| Ситуация                           | Корректировка                                               |
| ---------------------------------- | ----------------------------------------------------------- |
| Нет данных по партнёрам            | Сначала минимальная отчётность, затем управление по KPI     |
| Много вопросов по одному блоку     | Обновить базу знаний и провести отдельное обучение          |
| Низкая посещаемость обучений       | Перенести фокус на A/B-партнёров и персональные приглашения |
| Есть активность, но нет броней     | Разобрать аргументацию, условия, возражения и путь клиента  |
| Брони есть, но сделки тормозятся   | Разобрать сопровождение, документы, сроки и ответственность |
| Согласования СУ-10 тормозят запуск | Ввести недельный список обязательных решений                |

## Итоговые артефакты 90-го дня

- Итоговый отчёт по пилоту и 90-дневному циклу.
- Обновлённая ABC-карта партнёрской сети.
- Регламент партнёрского канала.
- База знаний, скриптов и шаблонов.
- Отчёт по агрегаторам.
- Рекомендации по масштабированию на другие объекты СУ-10.
- Решение по дальнейшему эксклюзиву, расширению или корректировке модели.
