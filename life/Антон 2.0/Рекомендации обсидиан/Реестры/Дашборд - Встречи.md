# Дашборд - Встречи

Минимальные поля для встреч:
`type:: meeting`, `date:: YYYY-MM-DD`, `client::`, `project::`, `next::`.

## Предстоящие
```dataview
TABLE date, client, project, next
FROM ""
WHERE type = "meeting" AND date >= date(today)
SORT date ASC
```

## Прошедшие
```dataview
TABLE date, client, project, next
FROM ""
WHERE type = "meeting" AND date < date(today)
SORT date DESC
```
