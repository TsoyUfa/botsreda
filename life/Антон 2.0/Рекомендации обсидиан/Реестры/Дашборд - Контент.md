# Дашборд - Контент

Минимальные поля для контента:
`type:: content`, `status:: idea|draft|ready|published`, `platform::`, `deadline:: YYYY-MM-DD`, `project::`, `next::`.

## В работе
```dataview
TABLE status, platform, deadline, project, next
FROM ""
WHERE type = "content" AND status != "published"
SORT deadline ASC
```

## Опубликовано
```dataview
TABLE platform, deadline, project
FROM ""
WHERE type = "content" AND status = "published"
SORT deadline DESC
```
