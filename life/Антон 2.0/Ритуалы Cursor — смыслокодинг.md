# Ритуалы Cursor — смыслокодинг

Минимальный набор привычек из практикума, адаптированный под Obsidian + Cursor.

---

## Ежедневно

### Утро (5–10 мин) — фокус

**Когда:** начало рабочего дня  
**Файлы:** профиль, инбокс, актуальный план в `Планы/`

```text
Прочитай: Антон 2.0 — профиль обо мне, Инбокс задач — май 2026, последний план в Планы/.
Цель 500k+ RUB/мес. Дай 1 рычаг на сегодня + 1 действие на 24ч.
Метки: Сегодня | На этой неделе | В бэклог.
```

**Skill:** `business-decision-router`

---

### Перед кодом / большой задачей — план

**Когда:** задача >1 часа или новый MVP  
**Куда:** `Антон 2.0/Планы/YYYY-MM-DD-название.md`

```text
Не пиши код. Research 15 мин → план с фазами [ ] → mvp-money-filter → покажи GO/NO-GO.
```

**Rules:** `delivery-to-result-flow`, `mvp-money-filter`  
**Skill:** `cursor-workflow-coach`

---

### Перед показом клиенту / деплоем — «10 причин»

```text
Стресс-тест: 10 причин, почему [оффер/демо/сайт] не купят или подведут.
Жёстко, без утешений. По каждой — фикс. Контекст: недвижимость, CRM, [продукт].
```

---

### После фазы — микро-QA

```text
Фаза [N] закончена.
1. Ошибки в том, что ты сделал в этой фазе.
2. Security review (ПДн, .env, фронт) — rule security-pd-review.
3. Обнови план в Планы/: [x] и промт следующей фазы.
```

---

### Вечер (5 мин) — ретроспектива

**Куда:** `Антон 2.0/retrospectives/YYYY-MM-DD_кратко.md` (шаблон ниже)

- Что сделано / не сделано  
- Один инсайт  
- Одно правило на будущее (в skill или заметку)

---

## Еженедельно

| День | Ритуал |
|------|--------|
| Понедельник | Утренний фокус + приоритет недели из инбокса |
| Среда | Прогресс по активному плану в `Планы/` — что срезать |
| Пятница | «Что двигает 500k+» — 3 проекта: оставить / пауза / убить |

```text
Какие активные планы и проекты реально двигают 500k+ за месяц?
Таблица: проект | вклад | следующий шаг | решение (продолжить/пауза/стоп).
```

---

## Когда контекст переполнен

```text
Напиши полный промт для нового чата: цель, сделано, файлы для чтения, следующий шаг, запреты.
```

Один чат = одна задача.

---

## Голос + скриншот

- Длинные ТЗ — голосом (быстрее в 3–4 раза)  
- UI/CRM/лендинг — скрин в чат + «что не так»  
- Mac: `Cmd+Shift+Ctrl+4` в буфер; Win: `Win+Shift+S`

---

## Чеклист «сессия смыслокодинга»

- [ ] Прочитан нужный контекст (@file / @folder), не весь vault  
- [ ] Есть план-файл для задачи >1ч  
- [ ] Пройден mvp-money-filter для новой идеи  
- [ ] «10 причин» перед внешним показом  
- [ ] Security после работы с формами/CRM/деплоем  
- [ ] Ретроспектива записана  
- [ ] Один следующий шаг на 24ч назван явно
