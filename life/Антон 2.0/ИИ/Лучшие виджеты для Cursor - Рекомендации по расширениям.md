# Лучшие виджеты для Cursor — Рекомендации по расширениям для предпринимателя

## Введение: Зачем нужны расширения для Cursor предпринимателю

Cursor — это мощный AI-ассистент для предпринимателей, построенный на базе VS Code. Одно из его главных преимуществ — полная совместимость с экосистемой расширений VS Code. Это означает, что вы можете использовать тысячи расширений для улучшения своего опыта работы с бизнес-задачами.

**Расширения (виджеты) помогают предпринимателю:**
- Организовать работу с документами и заметками (Markdown, PDF, таблицы)
- Управлять задачами и проектами (TODO-листы, закладки, планирование)
- Работать с данными (CSV, Excel, базы данных, CRM)
- Тестировать API и интеграции (REST API, веб-сервисы)
- Визуализировать информацию (диаграммы, графики, презентации)
- Автоматизировать рутинные задачи (экспорт, импорт, форматирование)
- Повысить продуктивность (быстрая навигация, поиск, структурирование)

**Результат:** Cursor превращается из AI-ассистента в мощную рабочую среду для предпринимателя, где можно работать с документами, данными, задачами и проектами в одном месте.

---

## Быстрый старт: Топ-5 расширений для начала

Если вы только начинаете использовать расширения, начните с этих пяти:

1. **Markdown All in One** — работа с документами (обязательно!)
2. **Todo Tree** — управление задачами
3. **Bookmarks** — быстрая навигация
4. **Markdown Preview Enhanced** — красивые документы и презентации
5. **Database Client** — работа с данными и CRM

Эти расширения покрывают 80% задач предпринимателя. Остальные добавляйте по мере необходимости.

## Как устанавливать расширения в Cursor

1. Откройте панель расширений: `Cmd+Shift+X` (Mac) или `Ctrl+Shift+X` (Windows/Linux)
2. Найдите расширение по названию
3. Нажмите "Install" (Установить)
4. Перезапустите Cursor при необходимости

**Альтернативный способ:**
- Откройте командную палитру: `Cmd+Shift+P` (Mac) или `Ctrl+Shift+P` (Windows/Linux)
- Введите "Extensions: Install Extensions"
- Найдите и установите нужное расширение

## Категория 1: Управление задачами и организация

### Todo Tree

**Что делает:**
- Находит все TODO, FIXME, NOTE, задачи в документах
- Отображает их в виде дерева в боковой панели
- Подсвечивает задачи в тексте

**Что улучшает в UX:**
- Быстрый доступ ко всем задачам в проекте
- Не теряете важные заметки и дела
- Организация работы над проектами

**Установка:**
```
Название: Todo Tree
ID: Gruntfuggly.todo-tree
```

**Базовые настройки:**
```json
{
  "todo-tree.regex.regex": "((//|#|<!--|;|/\\*|^)\\s*($TAGS)|^\\s*- \\[ \\])",
  "todo-tree.highlights.customHighlight": {
    "TODO": {
      "icon": "check",
      "type": "text",
      "foreground": "#ff6b6b"
    },
    "ВАЖНО": {
      "icon": "flame",
      "type": "text",
      "foreground": "#ffa500"
    },
    "ИДЕЯ": {
      "icon": "lightbulb",
      "type": "text",
      "foreground": "#4ec9b0"
    }
  }
}
```

**Примеры использования в Markdown:**
```markdown
- [ ] TODO: Встретиться с застройщиком
- [ ] ВАЖНО: Подготовить презентацию для клиента
- [ ] ИДЕЯ: Создать новый продукт для риэлторов
```

## Практические советы

### Как найти новые расширения

1. **Изучайте категории в Marketplace:**
   - Откройте панель расширений
   - Просматривайте популярные расширения по категориям
   - Читайте отзывы и рейтинги

2. **Следите за обновлениями:**
   - Регулярно проверяйте обновления расширений
   - Читайте changelog для новых возможностей

3. **Сообщества:**
   - Reddit r/vscode
   - GitHub Awesome VS Code
   - Блоги разработчиков

### Как кастомизировать под себя

1. **Настройте горячие клавиши:**
   - `Cmd+K Cmd+S` (Mac) / `Ctrl+K Ctrl+S` (Windows) — настройка клавиатуры
   - Назначьте удобные комбинации для часто используемых функций
   - Например: быстрый доступ к предпросмотру Markdown, создание задач, закладок

2. **Создайте свой набор расширений:**
   - Экспортируйте список расширений: `code --list-extensions`
   - Сохраните конфигурацию для синхронизации между устройствами
   - Документируйте, какие расширения для каких задач используете

3. **Настройте workspace:**
   - Создайте `.vscode/settings.json` для проектных настроек
   - Настройте `.vscode/extensions.json` для рекомендуемых расширений

**Пример `.vscode/extensions.json` для предпринимателя:**
```json
{
  "recommendations": [
    "yzhang.markdown-all-in-one",
    "shd101wyy.markdown-preview-enhanced",
```
