**Ты покупаешь не просто транскрибацию. Ты покупаешь систему управления продажами на основе данных.**

Полная автоматизация анализа всех звонков компании с глубокой аналитикой и готовыми промптами для работы.

**Что получаешь:**
- ✅ Транскрибированные диалоги всех отделов
- ✅ 5 готовых промптов для анализа
- ✅ Операционную систему в Telegram
- ✅ База данных всех разговоров
- ✅ Система взаимодействия и отчётов

**Результат:** Полный контроль над продажами через данные, а не интуицию.

## 📦 ЧТО ВХОДИТ В СИСТЕМУ

### 1. 📞 ТРАНСКРИБАЦИЯ ВСЕХ ЗВОНКОВ

**Что получаешь:**
- Полная транскрибация всех диалогов всех отделов
- Автоматическое определение ролей (эксперт/клиент)
- Разбивка на смысловые блоки
- Исправление терминов под нишу недвижимости

**Формат:**
- Текстовая расшифровка каждого звонка
- Структурированные данные
- Хранение в базе данных

**Результат:** Видишь каждый разговор в деталях. Ничего не теряется.

---

### 2. 🤖 5 ГОТОВЫХ ПРОМПТОВ ДЛЯ АНАЛИЗА

**Что получаешь:**
- 5 настроенных промптов для анализа звонков
- Готовые запросы для получения инсайтов
- Автоматический анализ по заданным критериям

**Что анализируют промпты:**
1. **Этап воронки продаж** — на каком этапе находится клиент
2. **Успешность этапа** — прошёл ли клиент этап успешно
3. **Качество общения** — как работает эксперт
4. **Реакции клиента** — позитив, негатив, вопросы, сомнения
5. **Следующие шаги** — назначена ли встреча, что дальше

**Результат:** Автоматический анализ каждого звонка без ручной работы.

---

### 3. 📱 ОПЕРАЦИОННАЯ СИСТЕМА В TELEGRAM

**Что получаешь:**
- Telegram-бот для доступа к аналитике
- Уведомления о важных звонках
- Быстрый доступ к данным
- Отчёты и дашборды

**Функционал:**
- Просмотр транскриптов звонков
- Аналитика по отделам и экспертам
- Тревожные сигналы (проблемные звонки)
- Ежедневные/еженедельные отчёты

**Результат:** Вся информация в одном месте. Доступ с телефона.

---

### 4. 💾 БАЗА ДАННЫХ

**Что получаешь:**
- Структурированная база всех разговоров
- Поиск по ключевым словам
- Фильтры по отделам, экспертам, датам
- Экспорт данных

**Формат:**
- Структурированное хранение
- Быстрый поиск
- Интеграция с CRM (при необходимости)

**Результат:** Вся история звонков в одном месте. Легко найти нужное.

---

### 5. 🔄 СИСТЕМА ВЗАИМОДЕЙСТВИЯ

**Что получаешь:**
- Автоматическая отправка данных в CRM
- Интеграция с существующими системами
- Настройка уведомлений
- Отчёты для руководства

**Интеграции:**
- CRM (AmoCRM, Битрикс24 и др.)
- Telegram для команды
- Email-отчёты
- Дашборды (при необходимости)

**Результат:** Данные автоматически попадают туда, где нужно.
