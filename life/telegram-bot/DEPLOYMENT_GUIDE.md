## 🎯 Быстрая инструкция по выгрузке

### Шаг 1: Выберите платформу

#### 🔹 Вариант 1: Netlify (рекомендую)
**Почему Netlify:**
- ✅ Бесплатно и надежно
- ✅ Автоматическое HTTPS (обязательно для Telegram)
- ✅ Глобальный CDN
- ✅ Простая настройка
- ✅ Поддержка custom domains

**Как выгрузить:**
1. Зайдите на [https://netlify.com](https://netlify.com)
2. Нажмите "Sign up" или "Log in"
3. "New site from Git" 
4. Подключите ваш GitHub аккаунт
5. Выберите репозиторий с проектом
6. Настройки сборки:
   - **Build command:** `echo 'Static site'`
   - **Publish directory:** `deploy-dist`
7. Нажмите "Deploy site"

#### 🔹 Вариант 2: Vercel
**Как выгрузить:**
1. Зайдите на [https://vercel.com](https://vercel.com)
2. "New Project" → "Import Git Repository"
3. Выберите репозиторий
4. Настройки:
   - **Framework Preset:** `Other`
   - **Output Directory:** `deploy-dist`
5. Нажмите "Deploy"

#### 🔹 Вариант 3: GitHub Pages
**Как выгрузить:**
1. Откройте ваш репозиторий на GitHub
2. Перейдите в Settings → Pages
3. Source: "Deploy from a branch"
4. Branch: `main` (или ваш основная ветка)
5. Directory: `/deploy-dist`
6. Нажмите "Save"

### Шаг 2: Получите URL сайта
После деплоя вы получите URL вида:
- Netlify: `https://random-name-123456.netlify.app`
- Vercel: `https://your-project.vercel.app` 
- GitHub Pages: `https://your-username.github.io/repo-name`

### Шаг 3: Обновите URL в боте
Откройте файл `bot.py` и замените:
```python
# Было:
url="https://your-web-app-url.com"

# Станет (пример для Netlify):
url="https://random-name-123456.netlify.app"
```

### Шаг 4: Перезапустите бота
```bash

```

### Шаг 5: Тестирование
1. Отправьте боту `/start`
2. Нажмите кнопку "🎓 Открыть курс в Web App"
3. Web App должен открыться в Telegram

## 🆘 Если что-то не работает

### Проблема: Web App не открывается в Telegram
**Решение:**
1. Проверьте, что URL начинается с `https://`
2. Убедитесь, что сайт доступен в браузере
3. Проверьте CORS заголовки
4. Перезапустите бота с новым URL

### Проблема: Видео не воспроизводятся
**Решение:**
1. Проверьте настройки приватности видео
2. Убедитесь, что видео разрешено для встраивания
3. Попробуйте другой видеохостинг

### Проблема: Файлы не скачиваются
**Решение:**
1. Проверьте настройки доступа Google Docs
2. Убедитесь, что ссылки публичные
3. Проверьте CORS на сервере
