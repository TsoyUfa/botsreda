# 🎓 Telegram Web App «Среда Обучение»

Полноценная система обучения для агентов по недвижимости на базе Telegram Web Apps (TWA) с методологией Р.О.С.Т.

## 🌟 Особенности

### 📱 Telegram Web App
- **Профессиональный интерфейс** внутри Telegram
- **Один клик для входа** без логинов/паролей
- **Адаптивный дизайн** для мобильных устройств
- **Чувство дорогого мобильного приложения**

### 📚 Система обучения
- **7 модулей** по методологии Р.О.С.Т.
- **Видеоуроки** с поддержкой YouTube/VK/Kinescope
- **PDF/Excel материалы** для скачивания
- **Тесты и задания** с ИИ-проверкой
- **Прогресс обучения** с подробной статистикой

### 👨‍🏫 Для преподавателя
- **Команда `/monitor`** для полного контроля
- **Отслеживание всех студентов**
- **История ответов и голосовых сообщений**
- **Детальная аналитика** и статистика
- **Готово к монетизации** в будущем

## 🚀 Быстрый старт

### 1. Запуск бота
```bash
# Установка зависимостей
pip3 install -r requirements.txt

# Запуск Telegram бота
python3 bot.py
```

### 2. Web App
Web App автоматически разворачивается через GitHub Pages или Netlify.

После деплоя обновите URL в файле `bot.py`:
```python
# Замените на ваш URL после деплоя
url="https://tsoyufa.github.io/botsreda"  # GitHub Pages
# или
url="https://botsreda-netlify.app"        # Netlify
```

## 🔧 Техническая реализация

### Стек технологий
- **Telegram Bot API** (aiogram 3.x)
- **Telegram Web Apps** (нативная интеграция)
- **Flask** (для API сервера)
- **SQLite** (хранение данных)
- **YAML** (управление контентом)

### Структура проекта
```
telegram-bot/
├── bot.py              # Основной файл бота
├── config.py           # Конфигурация модулей
├── lessons.py          # Данные уроков
├── db.py               # Работа с базой данных
├── webapp/             # Telegram Web App
│   ├── index.html      # Главный интерфейс
│   ├── styles.css      # Стили (Telegram Theme)
│   ├── app.js          # JavaScript логика
│   └── content.yml     # Контент модулей
├── deploy-dist/        # Файлы для деплоя
├── webapp_server.py    # API сервер (опционально)
└── webapp_manager.py  # Управление контентом
```

## 🌐 Деплой

### GitHub Pages (просто)
1. Файлы уже подготовлены в `deploy-dist/`
2. В настройках репозитория: Settings → Pages
3. Source: Deploy from branch → main → `/deploy-dist`
4. Сайт будет доступен по: `https://tsoyufa.github.io/botsreda`

### Netlify (рекомендуется)
1. Зайдите на [netlify.com](https://netlify.com)
2. "New site from Git" → выберите этот репозиторий
3. Build settings:
   - Build command: `echo 'Static site'`
   - Publish directory: `deploy-dist`
4. Деплой и красивый URL

## 📱 Использование

### Для студентов
1. Отправьте боту `/start`
2. Нажмите кнопку **"🎓 Открыть курс в Web App"**
3. Изучайте материалы, смотрите видео, скачивайте файлы
4. Выполняйте задания и проходите тесты

### Для преподавателя
1. Отправьте боту `/monitor`
2. Просматривайте прогресс всех студентов
3. Анализируйте ответы и активность
4. Получайте детальную статистику
