# 🎓 Telegram Web App LMS для «Среда обучения»

## 📋 Обзор проекта

Создаем полноценную систему обучения (LMS) через Telegram Web Apps, которая будет работать как профессиональное приложение внутри вашего Telegram-бота.

## 🎯 Функциональность

### Для студентов:
- 🔥 **Полноценный интерфейс** внутри Telegram (Web App)
- 📚 **Персональный кабинет** с прогрессом обучения
- 🎥 **Интегрированный видеоплеер** (без перехода на сторонние сайты)
- 📄 **Просмотр PDF/Excel** прямо в Telegram
- ✅ **Тесты и домашние задания** с мгновенной проверкой
- 💳 **Бесшовная оплата** модулей через Telegram Payments

### Для вас как преподавателя:
- 📊 **Полная аналитика** по каждому студенту
- 📝 **Управление контентом** через админ-панель
- 💰 **Автоматизация продаж** и выдачи доступа
- 🎙️ **Сбор голосовых ответов** и обратной связи
- 📈 **Прогресс каждого студента** в реальном времени

## 🏗️ Архитектура системы

### Компоненты:
1. **Telegram Bot** - основной интерфейс и вход в систему
2. **Web App Frontend** - полноценный интерфейс обучения
3. **Backend API** - управление данными и интеграции
4. **База данных** - хранение прогресса, ответов, платежей
5. **Видеохостинг** - интеграция с VK Видео/Kinescope
6. **Файловое хранилище** - Google Диск для PDF/Excel
7. **Платежная система** - Telegram Payments + Webhooks

### 2. Запуск бота с Web App
```bash

pip3 install fastapi uvicorn python-multipart

# Запуск API сервера
cd api
python3 main.py

# Запуск бота
cd ..
python3 bot.py
```

### 3. Тестирование Web App
1. Отправьте боту `/start`
2. Нажмите кнопку "Обучение"
3. Web App откроется внутри Telegram
