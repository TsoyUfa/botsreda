# 🎓 Telegram-бот «Среда Обучение» — Полное руководство

## 📋 О проекте

Бот создан для обучения агентов по недвижимости методологии Р.О.С.Т. с полным отслеживанием прогресса и ответов студентов.

## ✅ Основные возможности

### 1. 🎓 Обучение
- **7 бесплатных блоков** по методологии Р.О.С.Т.
- **Интерактивные тесты** с выбором вариантов и текстовыми ответами
- **ИИ-проверка** текстовых ответов через Gemini API
- **Аудио-материалы** для каждого блока

### 2. 📊 Мониторинг студентов
- **Полное отслеживание** каждого студента
- **История всех ответов** (текстовые, выбор варианта, голосовые)
- **Прогресс по модулям** в реальном времени
- **Статистика активности** и времени обучения

### 3. 🎙️ Работа с голосовыми сообщениями
- **Автоматическая запись** всех голосовых сообщений студентов
- **Сохранение длительности** и file_id для доступа
- **Логирование** в базе данных для анализа

### 4. 🔍 Админ-панель мониторинга
Команда `/monitor` открывает панель с разделами:

- **📊 Дашборд** - ключевая статистика по всем студентам
- **👥 Все студенты** - список с фильтрацией по активности
- **🔍 Поиск студента** - по ID, имени или username
- **📝 Новые ответы** - последние 15 ответов студентов
- **🎙️ Голосовые сообщения** - все голосовые сообщения
- **📈 Активность сегодня** - статистика за текущий день

### 5. 📤 Экспорт данных
- **Экспорт данных студента** в формат JSON
- **История ответов** с ИИ-оценками
- **Полная статистика** по обучению

## 👨‍💻 Использование для преподавателя

### 1. Мониторинг студентов
- Отправьте боту команду `/monitor`
- Нажмите "👥 Все студенты" чтобы увидеть список
- Выберите студента для детальной информации

### 2. Просмотр ответов
- В детальной информации студента нажмите "📝 Все ответы"
- Вы увидите:
  - Текстовые ответы с ИИ-проверкой
  - Выбор вариантов в тестах
  - Голосовые сообщения
  - Оценки и прогресс

### 3. Голосовые сообщения
- Студенты могут отправлять голосовые сообщения боту
- Все сообщения автоматически сохраняются
- Доступна длительность и время отправки

### 4. Экспорт данных
- В детальной информации студента нажмите "📤 Экспорт данных"
- Вы получите JSON файл с:
  - Полной историей ответов
  - Прогрессом по модулям
  - Статистикой обучения
  - Временем в системе

## 📊 Структура данных

### Таблица student_responses:
```
- user_id: ID студента
- module_id: Номер блока обучения
- question_type: Тип вопроса ('choice', 'free_text', 'voice')
- question: Текст вопроса
- answer: Ответ студента
- is_correct: Правильность ответа
- score: Оценка (для текстовых ответов)
- ai_feedback: Обратная связь от ИИ
- voice_file_id: ID голосового файла
- voice_duration: Длительность голосового сообщения
- created_at: Время ответа
```

## 🔧 Настройка и кастомизация

### 1. Изменение блоков обучения

В файле `bot.py` функция `evaluate_answer_with_ai`:
- Измените промпт для ИИ
- Настройте порог оценки
- Измените формат обратной связи

### 3. Админ-панель
Файлы для настройки:
- `simple_admin.py` - основная панель
- `extended_monitoring.py` - дополнительные функции
- `student_monitoring.py` - логирование ответов

## 🐛 Решение проблем

### 1. Бот не запускается
- Проверьте все токены в `.env` файле
- Убедитесь, что все зависимости установлены
- Проверьте путь к базе данных

### 2. Не работает ИИ-проверка
- Проверьте `GEMINI_API_KEY` в `.env`
- Убедитесь, что у вас есть доступ к Gemini API
- Проверьте интернет-соединение

### 3. Нет доступа к админ-панели
- Убедитесь, что ваш Telegram ID указан правильно
- Проверьте, что вы администратор в коде
- Перезапустите бота после изменений
