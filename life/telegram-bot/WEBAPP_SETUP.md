## 🚀 Быстрый запуск

### Шаг 1: Запуск Web App сервера

```bash
cd /Users/anton_tsoy/Desktop/Обсидиан/life/telegram-bot

# Установка зависимостей (если еще не установлены)
pip3 install -r requirements.txt

# Запуск сервера Web App
python3 webapp_server.py
```

После запуска сервер будет доступен по адресу:
- **Web App:** http://localhost:5000
- **API:** http://localhost:5000/api
- **Карта сайта:** http://localhost:5000/api/sitemap

### Шаг 2: Настройка URL для Web App

В файле `bot.py` замените `https://your-web-app-url.com` на ваш URL:

**Для локального тестирования:**
- Используйте `http://localhost:5000` (но Telegram требует HTTPS)
- **Или используйте ngrok для создания туннеля:**
  ```bash
  # Установите ngrok: brew install ngrok
  # Создайте туннель для локального сервера
  ngrok http 5000
  ```
- Вы получите URL вида `https://abc123.ngrok.io`

**Для продакшена:**
- Разместите файлы Web App на любом хостинге (Vercel, Netlify, GitHub Pages)
- Или используйте свой сервер с Nginx/Apache

### Шаг 3: Запуск бота

```bash
# В новой вкладке терминала
cd /Users/anton_tsoy/Desktop/Обсидиан/life/telegram-bot

# Запуск бота
python3 bot.py
```

## 🔧 Настройка контента

### Структура файлов контента:
```
telegram-bot/
├── webapp/
│   ├── index.html          # Основной интерфейс
│   ├── styles.css          # Стили
│   ├── app.js             # JavaScript логика
│   └── content.yml        # Контент курсов
├── webapp_manager.py      # Менеджер контента
├── webapp_server.py      # Сервер для Web App
└── bot.py               # Telegram бот
```

### Редактирование контента:
1. Откройте файл `webapp/content.yml`
2. Измените структуру модулей и уроков
3. Перезапустите сервер: `python3 webapp_server.py`

### Добавление видео:
- **YouTube:** Добавьте ссылку на видео
- **VK Видео:** Добавьте ссылку на видео
- **Kinescope:** Добавьте ссылку на видео

### Добавление файлов:
- Загрузите файлы в Google Docs/Drive
- Настройте доступ "Просмотр для всех"
- Добавьте ссылки в `content.yml`

## 🐛 Решение проблем

### Web App не открывается:
1. Проверьте, что сервер запущен: `python3 webapp_server.py`
2. Проверьте URL в боте: должен быть HTTPS
3. Для локальной разработки используйте ngrok

### Видео не отображается:
1. Проверьте ссылки в `content.yml`
2. Убедитесь, что видео доступно для встраивания
3. Для YouTube: проверьте настройки приватности

### Файлы не скачиваются:
1. Проверьте настройки доступа Google Docs
2. Убедитесь, что ссылки корректны
3. Для больших файлов используйте прямые ссылки
