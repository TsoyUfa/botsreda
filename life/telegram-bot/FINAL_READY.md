## 🚀 Вам нужно сделать только 3 простых шага:

### Шаг 1: Развернуть Web App (2 минуты)
**Быстрый вариант - Netlify:**
1. Откройте: **https://app.netlify.com**
2. **"New site from Git"** → выберите **botsreda**
3. Настройки:
   - **Build command:** `echo 'Static site'`
   - **Publish directory:** `deploy-dist`
4. Нажмите **"Deploy site"**

### Шаг 2: Обновить URL в боте (1 минута)
1. Откройте файл: `bot.py`
2. Найдите строку: `url="https://your-web-app-url.com"`
3. Замените на ваш URL (например: `https://botsreda-netlify.app`)
4. Сохраните файл
