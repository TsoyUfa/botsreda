## 📋 Детальная инструкция для GitHub Pages:

**Когда git push завершится:**

1. **Открыть:** https://github.com/TsoyUfa/botsreda
2. **Settings** → **Pages**
3. **Source:** Deploy from a branch
4. **Branch:** main
5. **Directory:** /deploy-dist
6. **Save**
7. **Через 1-2 минуты:** сайт будет доступен

**Если нет опции /deploy-dist:**
```bash
# Альтернатива: переименовать в docs
cd "/Users/anton_tsoy/Desktop/Обсидиан/life/telegram-bot"
mv deploy-dist docs
git add .
git commit -m "Rename to docs for GitHub Pages"
git push origin main
```
Тогда выбрать **Directory: /docs**
