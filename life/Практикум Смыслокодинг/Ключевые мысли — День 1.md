# Ключевые мысли — День 1: Что такое смыслокодинг и как начать

> Источник: транскрибация живого эфира Артемия Миллера (предприниматель, резидент Сколково, сеть клиник Миссис Лазер, AI-телефония Аишка).

---

## 1. Смыслокодинг vs Вайпкодинг

- **Вайпкодинг** = быстро написать код (для разработчиков)
- **Смыслокодинг** = переложить свою экспертизу/смыслы в продукт через AI (для всех)
- Цель: **"Умеешь думать + AI = делаешь продукты"**

---

## 2. Рекомендуемый стек

| Компонент | Рекомендация | Почему |
|-----------|-------------|--------|
| **Редактор** | VS Code | Бесплатный, гибкий, один центр управления |
| **Модель** | Claude Opus 4.7 | 1 млн токенов, лучший по бенчмаркам |
| **Альтернатива** | ChatGPT 5.5/Codex | 250K токенов, для поиска ошибок |
| **Китайская альтернатива** | DeepSeek V4 Pro | 1 млн токенов, если Claude заблокирован |

**НЕ использовать:**
- Cursor — дорого, форк VS Code
- Bolt / Lovable / Replit — "песочница для детей", в 10-20 раз дороже

## 4. Трансформация эффективности

- **X10**: ты + второй мозг + AI-клон (уже сейчас)
- **X100**: армия агентов, сами находят проблемы и решения
- **X1000**: рой агентов, полная автономия бизнеса ("я-компания")

---

## 6. Методология работы

> Сначала сделай так, чтобы работало и приносило пользу. Всё остальное потом.

- **Маленькие задачи**: идея → промт → код → запуск
- **Большие проекты**: планирование → фазы → CLAUDE.md → база знаний → итерации
- Не пихать всё в одно контекстное окно — рубить маленькими кусочками

---

## 7. Ключевые инструменты

- **Bypass Permissions** в Claude Code — режим полной автономии
- **WhisperFlow** — голосовой ввод на русском языке (бесплатная локальная модель)
- **Многоагентная система** — параллельные агенты для ресёрча и задач
- **CLAUDE.md** — файл-инструкция с памятью проекта (подробно на День 2)
- **Папка бизнес** — база знаний проекта (подробно на День 2)

---

## 8. Почему у большинства не получается

- Расплывчатый запрос — "сделай сайт" = мусор
- Хотят всё сразу одним промтом
- Боятся ошибок и бросают
- Не используют методологию (планы, фазы, итерации)

---

## 9. Обход блокировок из России (3 пути)

1. **Путь 1**: Старый Gmail + VPN + ESIM одной страны + иностранная карта → подписка
2. **Путь 2**: Удалённый сервер в США ($5-10/мес) + Claude Code терминальный + подключение через VS Code по SSH
3. **Путь 3**: Российский посредник (Звено AI) — дороже, но за рубли

---

## 10. N8N устарел

- Вайпкодинг кратно эффективнее любых low-code платформ
- N8N = "как с дедуном договариваться" — не гибко, костыльно для больших задач

---

## 11. Ключевой принцип дня

> "Кодинг был первым шагом. Смыслокодинг — следующий."

- Каждый день твой проект работает лучше, чем вчера
- Всё, что ты навайпкодил — работает сегодня (накопительный эффект)
