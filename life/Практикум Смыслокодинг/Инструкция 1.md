## Инструкция

[Транскрипция эфира](https://smyslokod.ru/dashboard/praktikum-may-2026/transcription/f8e6e9e3-d91f-42a0-9401-426107f645e4)[Презентация эфира](https://storage.yandexcloud.net/smyslocode-uploads/presentations/1778621121939-f48b2761.pdf)

### Что получится после инструкции

VS Code + Claude Code + платная подписка Claude, настроенные на максимум. Голосовой ввод и скриншоты подключены. Окружение разработчика (Node.js, pnpm, Git, расширения VS Code) поставлено.

**Время:** 60-90 минут. Большую часть просто ждёшь, пока скачается.

**Железо:** Mac M2/M3/M4 идеально. Mac M1 - работает, но греется. Windows - 16 ГБ RAM комфортно, 8 ГБ - впритык.

**Как работать с инструкцией:** ниже 6 этапов — это твой план дня. Нажми на стрелку слева от заголовка этапа, чтобы развернуть пошаговые действия. Раскрывай по одному, делай, сворачивай, переходи к следующему. Если что-то не пошло — внутри каждого этапа есть блок «если не получилось».

Этап 1. Установи VS Code · ~5 мин▸

1. Зайди на [code.visualstudio.com](https://code.visualstudio.com/), нажми **Download**
2. Установи: двойной клик → «Далее» → «Готово»
3. Запусти VS Code

**Создай папку проекта:**

- **File → Open Folder** (Mac: `Cmd + O`, Win: `Ctrl + K, Ctrl + O`)
- Создай новую папку на рабочем столе, например `my-project`
- Выбери её - VS Code откроет проект

Этого достаточно. С остальным VS Code (терминалы, шорткаты, панели) разберёшься по ходу — Claude Code сам будет всё запускать и показывать, ты лишь нажимаешь «Разрешить».

Этап 2. Зарегистрируй аккаунт Claude · ~15-30 мин▸

#### Шаг 2.1. Подготовь Gmail и VPN

**Требования к Gmail:**

- Возраст почты - минимум **1 год**
- Страна аккаунта - **США**
- Подтверждённый возраст 18+ в настройках Google

**Если страна Gmail не США:**

- Открой [payments.google.com](https://payments.google.com/) под включённым VPN США
- Настройки → Изменить страну → США
- ⚠️ Смена доступна **только раз в год** - выбирай осознанно

**VPN:**

- Используй платный VPN со **стабильным статическим US IP** (не прыгающий между странами)
- Старайся не выключать VPN на компе, где открыт Claude. Если выключал — перед открытием Claude включи обратно. Блокировка идёт за RU IP в момент входа, а не за то, что иногда VPN был выключен.
- Не ставь мобильное приложение Claude
- Не заходи в Claude с других браузеров/устройств

#### Шаг 2.2. Зарегистрируйся через Google (без SMS)

2. Открой [claude.ai](https://claude.ai/)
3. Нажми **Continue with Google**
4. Выбери свой Gmail

**Если телефон не запросили - готово, переходи к Этапу 3.**

**Вариант В - готовый аккаунт через посредника:**

Этап 3. Оплати подписку · ~10 мин▸

#### Шаг 3.1. Прогрев - три ступени

⚠️ Если сразу прыгнешь на $100 - заблокирует 100%, даже с идеальной картой и VPN.

1. **3-5 дней** просто пользуйся [claude.ai](https://claude.ai/) бесплатно (любые вопросы, переводы, рецепты)
2. Купи **Pro $20**, поработай день-два
3. Только когда упёрся в лимит $20 - переходи на **Max x5 $100** или **Max x20 $200**

#### Шаг 3.2. Выбери тариф

- **Pro $20/мес** - только для разогрева
- **Max x5 $100/мес (~₽9 000)** - рабочий тариф
- **Max x20 $200/мес (~₽18 000)** - для нескольких параллельных проектов

#### Шаг 3.3. Подготовь карту

> ⚠️ **Важно прочитать до конца, прежде чем что-то выбирать**

>

> Все способы оплаты ниже - это наблюдения сообщества вайб-кодеров, которым делятся участники потоков. Это не мои личные рекомендации, я не получаю никаких комиссий от этих сервисов.

>

> Любой такой сервис может в любой момент: - перестать работать - попасть в чёрный список Anthropic (твой аккаунт забанят при оплате) - закрыться или обмануть - изменить условия и комиссии

>

> Все финансовые риски и ответственность за выбор - на тебе. Перед оплатой больших сумм бери минимальный план ($20), проверяй что связка работает, и только потом доплачивай.

**🥇 Что Claude принимает спокойнее всего**

- Личная физическая карта зарубежного банка в стране без санкций (Казахстан, Армения, Грузия, Таджикистан, ОАЭ, Турция). Если есть свой счёт в одной из этих стран или близкий человек с такой картой - используй её.
- Страна карты совпадает со страной Gmail и со страной VPN. Пример: gmail US + VPN US - желательно и карта американская. Если карта KZ - аккаунт и VPN тоже KZ. Совпадение страны во всех трёх местах снижает риск бана.
- Apple Pay через Apple ID той же страны, с балансом, пополненным легальной зарубежной картой. Чуть надёжнее прямой оплаты картой.

**🥈 Если своей зарубежной карты нет - сервисы виртуальных карт**

Виртуальная карта от стороннего сервиса - самый доступный путь. Список ниже - сервисы, которыми пользовались участники наших потоков. Это не оценка и не рейтинг, просто варианты, где можно завести карту:

- [platipomiru.com](https://platipomiru.com/) - ООО «КАПИБАРА», ИНН 9703231520. Выпускают виртуальные карты иностранных банков, у них есть отдельная инструкция по оплате Claude. Карта для подписок - ₽2 990 (оплата в евро), карта для путешествий - ₽3 990 (оплата в долларах). Пополнение через СБП. Заходить только через основной домен - в интернете есть поддельные сайты-двойники.
- Другие варианты - спрашивай в знакомых сообществах вайб-кодеров и ИИ-практиков. Конкретные имена я тут не пишу намеренно: связки меняются месяц к месяцу, что работало в апреле, в мае может банить.

Не плати сразу $100 или $200. Сначала бери $20 Pro и смотри сутки - не словишь ли бан. Если тихо - доплати до Max. В худшем случае теряешь $20, а не $200.

**❌ Что точно НЕ работает (объективные факты, не мнение)**

- Российские карты в системе Stripe. Stripe - платёжная система Anthropic. Карты любых РФ-банков она отклоняет на своём уровне, ещё до Anthropic.
- Предоплаченные Visa Gift и подарочные карты из массовых магазинов и от перекупов. Anthropic пересмотрел политику по ним весной 2026: при оплате блокируется аккаунт, деньги возвращаются, аккаунт мёртв.
- «Свежий аккаунт + первая оплата сразу на $100 + виртуальная карта». Бан почти всегда, даже если карта идеальная. Сам факт «новый аккаунт + крупная первая оплата + посредник» - красный флаг для защиты от мошенничества.

**🔑 Что важнее самой карты**

1. Прогрев аккаунта 3-5 дней. Свободное использование claude.ai, потом $20 Pro, и только после упирания в лимит - Max.
2. Не дёргать VPN в момент оплаты. Не переключать страну. Стабильность IP в момент платежа важнее самой карты.
3. Старый Gmail (год+) часто решает сильнее, чем свежий «правильный».
4. Один способ оплаты от регистрации до Max. Не мешать карту с подарочными кодами в одном аккаунте.

Этап 4. Поставь Claude Code в VS Code · ~5 мин▸

#### Шаг 4.1. Установи Node.js

1. [nodejs.org](https://nodejs.org/) → скачай **LTS**
2. Установи: «Далее» → «Далее» → «Готово»

#### Шаг 4.2. Установи расширение Claude Code

1. В VS Code → `Cmd / Ctrl + Shift + X`
2. В поиске напиши **claude**
3. Выбери **официальное расширение от Anthropic** (verified)
4. Нажми **Install**
5. Нажми на иконку Claude слева в сайдбаре
6. Откроется браузер → авторизуйся в Claude-аккаунте

Если расширение не пошло — поставь через терминал▸

Открой терминал прямо в VS Code: меню **View → Terminal** (внизу появится чёрное окно).

Вставь в него:

npm install -g @anthropic-ai/claude-code

📋 Копировать

Дождись окончания установки, потом запусти:

claude

📋 Копировать

#### Шаг 4.3. Настрой 5 параметров на максимум

Шестерёнка ⚙️ в панели Claude Code:

|Параметр|Значение|
|---|---|
|Permission Mode|**Bypass Permissions**|
|Default Mode|**Bypass Permissions**|
|Effort|**Max**|
|Thinking|**On**|
|Model|**Claude Opus 4.7**|

Если в меню нет «Bypass Permissions» или «Opus 4.7»▸

1. Открой **Settings** (та же шестерёнка)
2. Найди чекбокс **«Allow Bypass Permissions mode»** → включи
3. Закрой VS Code полностью (`Cmd / Ctrl + Q`)
4. Открой заново - настройки появятся

#### Шаг 4.4. Проверь лимиты

В Claude Code набери `/usage` или нажми **Account Usage** - покажет сожжённые токены и остаток до сброса.

Опционально: Codex от OpenAI параллельно▸

Если есть подписка ChatGPT, поставь параллельно:

1. Extensions → найди **«Codex»** (verified от OpenAI) → Install
2. Авторизуйся через ChatGPT-аккаунт
3. Настройки:

|Параметр|Значение|
|---|---|
|Permission Mode|**Full Access**|
|Effort|**Extra High**|
|Model|**GPT-5.5**|
|Mode|**Standard** (не Fast)|

Этап 5. Подключи голос и скриншоты · ~10-30 мин▸

#### Шаг 5.1. Голосовой ввод

Я диктую промпты голосом — в 3-4 раза быстрее, чем печатать. Сам пользуюсь локальным Whisper: бесплатно, на компе, ничего не уходит в облако.

Сейчас покажу метод смысло-кодинга в действии: **ты ничего не будешь ставить руками, ни одной команды в терминал.** Просто скопируешь 2 текста в чат Claude Code и будешь жать кнопку «Разрешить». Всё остальное он сделает сам.

**Перед началом:** открой VS Code в соседнем окне рядом с этим уроком. Будешь переключаться между этим окном (читаешь шаги) и VS Code (делаешь). Держи их рядом.

**Шаг 1. Открой чат Claude Code**

В левой панели VS Code есть колонка с иконками (File Explorer, Search, Git, Extensions и т.д.). Найди там иконку с буквой **C** (логотип Claude) — обычно она ниже значка магазина расширений. Кликни.

Справа откроется боковая панель — это чат с Claude.

**Шаг 2. Попроси Claude поставить скилл**

Скопируй текст ниже целиком (выдели весь блок, `Cmd/Ctrl + C`) и вставь в чат Claude:

ПРОМПТ ДЛЯ АГЕНТА📋 Копировать промпт

Поставь мне скилл для голосовой диктовки из репозитория https://github.com/Mobiss11/Whisper-Skill Сделай так, чтобы он был доступен глобально для Claude Code (папка скиллов в моём домашнем каталоге). Если у меня не установлен git — поставь его тоже. Если что-то не получится — объясни простыми словами, что мне нужно сделать.

Claude начнёт выполнять команды. Каждый раз, когда он попросит разрешение, нажимай **«Разрешить»** (Allow).

Если случайно нажал «Отказать» — просто напиши в чат: «повтори последнюю команду», и он попросит снова.

**Шаг 3. Открой новый чат**

В верхней части панели Claude Code справа есть кнопка **«+»** — это новый чат. Жми. Старый чат не удалится, он останется в истории (слева кнопка со списком).

Зачем новый чат: Claude видит свежеустановленные скиллы только при старте новой сессии. В старом чате он скилл ещё не «видит».

**Шаг 4. Попроси поставить голосовой ввод**

Скопируй текст ниже целиком и вставь в новый чат:

ПРОМПТ ДЛЯ АГЕНТА📋 Копировать промпт

Используй скилл whisper-skill. Сначала подтверди одной фразой, что ты его видишь в списке доступных скиллов. Если не видишь — скажи мне, я перезапущу окно. Если видишь — определи моё железо (ОС, процессор, память), подбери оптимальный локальный Whisper под него, поставь всё что нужно, настрой push-to-talk диктовку с глобальным хоткеем Ctrl+Shift+Space. В конце прогони тест: запиши короткую фразу через хоткей и покажи, что текст распознался.

Дальше — снова жми «Разрешить» на каждый запрос. Времени займёт **5-30 минут**, в основном на скачивание модели (~800 МБ через VPN). Можешь спокойно сходить за чаем.

**Шаг 5. Проверь работу**

Когда Claude напишет, что всё готово:

1. Встань курсором в любое поле ввода — хоть в чат Claude, хоть в файл VS Code, хоть в браузерную строку
2. Зажми `Ctrl + Shift + Space` (на Mac тоже Ctrl, не Cmd)
3. Скажи вслух: «напиши hello world на python»
4. Отпусти клавиши
5. Текст должен сам появиться в том поле, где стоял курсор

Если хоткей не работает — скопируй это в чат Claude▸

ПРОМПТ ДЛЯ АГЕНТА📋 Копировать промпт

Хоткей Ctrl+Shift+Space не работает: я его нажимаю, говорю, отпускаю — текст не появляется. Проверь, что именно сломано (нет прав доступа к микрофону, нет Accessibility-разрешения, конфликт хоткея с системой, модель не скачалась) и проведи меня по шагам к починке.

Это закроет 95% проблем. На Mac чаще всего — Accessibility-разрешение (нужно дать VS Code разрешение в системных настройках), Claude проведёт за руку.

Если хочешь готовое платное приложение▸

|Приложение|Цена|Платформа|
|---|---|---|
|[wisprflow.ai](https://wisprflow.ai/)|$12-15/мес|Mac + Windows|
|[superwhisper.com](https://superwhisper.com/)|$10/мес|Mac|

По функциям то же самое, что локальный Whisper. Просто красивый UI. Я лично не плачу — локальный делает всё то же.

Системный fallback (если вообще ничего не пошло)▸

- Mac: `System Settings → Keyboard → Dictation → On`
- Windows: `Win + H`

Работает прямо сейчас, без установки. Качество хуже, но писать голосом можно — не блокирует движение по практикуму.

#### Шаг 5.2. Скриншоты в буфер обмена

**Mac - системный шорткат:** `Cmd + Shift + Ctrl + 4` → выдели область → автоматически в буфер

Mac — удобнее перенастроить на одну руку▸

1. System Settings → Keyboard → Keyboard Shortcuts → Screenshots
2. Найди **«Copy picture of selected area to the clipboard»**
3. Кликни на текущую комбинацию → нажми **`Cmd + 4`** (или `Cmd + Shift + 4`)
4. Закрой настройки

**Windows:** `Win + Shift + S` → выдели область → готово

**Проверка:**

1. Сделай скриншот
2. Открой Claude Code, поле ввода
3. `Cmd / Ctrl + V` → скриншот должен появиться в чате

Этап 6. Запусти агента на установку окружения · ~15 мин▸

#### Шаг 6.1. Вставь промпт в Claude Code

ПРОМПТ ДЛЯ АГЕНТА📋 Копировать промпт

Я полный новичок в программировании, это мой первый опыт. Я прохожу практикум по смысло-кодингу. Моя задача сегодня - подготовить компьютер к разработке веб-сайтов на Next.js. Пожалуйста, сделай следующее: 1. Проверь мою операционную систему (Mac или Windows). 2. Проверь, что уже установлено: Node.js, pnpm, Git. Покажи версии. 3. Установи то, чего не хватает: - pnpm (npm install -g pnpm) - Git (если его нет) 4. После установки проверь: node -v, pnpm -v, git --version. 5. Помоги установить расширения VS Code: ESLint, Prettier, Tailwind CSS IntelliSense, Prisma. Правила: - Перед каждой командой объясняй простыми словами, что собираешься делать. - Жди моего разрешения перед запуском. - Если ошибка - объясни человеческим языком, предложи решение. - В конце скажи: «Окружение готово, можно начинать практикум».

#### Шаг 6.2. Подтверждай команды

Claude будет писать:

> Я хочу выполнить `npm install -g pnpm`. Это установит менеджер пакетов. Можно?

Жми кнопку **Run** или подтверждай словом. Агент выполнит, прочитает результат, пойдёт дальше.

⚠️ Если Claude пишет «Жду решение - применять или правки?» - **обязательно отвечай**. Без ответа он остановится навсегда. Каждые 5-10 минут проверяй чат.

#### Шаг 6.3. Где хранить API-ключи

Когда позже подключишь сторонние сервисы (платежи, AI-провайдеры, БД) - они дадут API-ключ.

1. В корне проекта создай файл `.env`
2. Клади ключи в формате:

`` `OPENROUTER_KEY=sk-abc123...`  ``

1. Скажи Claude:

> «Все секреты в `.env`. Файл в gitignore. Тебе читать его не надо - используй `process.env.OPENROUTER_KEY`.»

⚠️ Для разовых тестов с лимитом $5 на ключе - можно вставить прямо в чат. Боевые ключи - только через `.env`.

ПРОМПТ ДЛЯ АГЕНТА📋 Копировать промпт

ПРОМПТ ДЛЯ АГЕНТА📋 Копировать промпт

Продвинутый путь — Claude Code на своём сервере США (для второй недели)▸

Сложнее. Используй после первой недели локальной работы.

#### Что понадобится

- VPN США (см. Этап 2)
- Карта США (см. Этап 3)
- Старый Gmail с US-страной (см. Этап 2.1)
- Сервер: [hostkey.ru](https://hostkey.ru/), **₽1 110/мес** за сервер с US IP, оплата с РФ карты. На hostkey сейчас часто видны только RU-локации, тогда возьми [Hetzner](https://hetzner.com/) или [DigitalOcean](https://digitalocean.com/) — там US доступны.
- Антидетект-браузер: [donutbrowser.com](https://donutbrowser.com/ru)
- Номер: eSIM Tello/US Mobile или [onlinesim.io](https://onlinesim.io/)

#### Шаг 1. Закажи сервер

1. Открой провайдера → VPS → выбери план с US IP
2. ОС: **Ubuntu 24 LTS**
3. **Сразу добавь свой публичный SSH-ключ** (тогда пароль root не понадобится)
4. Оплати

#### Шаг 2. Подключись по SSH

В терминале (Mac Terminal / Windows PowerShell):

ssh root@IP_СЕРВЕРА

📋 Копировать

#### Шаг 3. Поставь зависимости

Выполни по очереди:

apt update && apt upgrade -y

📋 Копировать

curl -fsSL https://deb.nodesource.com/setup_20.x | bash -
apt install -y nodejs

📋 Копировать

npm install -g @anthropic-ai/claude-code

📋 Копировать

#### Шаг 4. Авторизуй Claude

claude login

📋 Копировать

1. Скопируй ссылку, которую выдал терминал
2. Вставь её в антидетект-браузер (с включённым VPN США, где ты регистрировал Claude)
3. Нажми **Authorize**
4. Скопируй код, вставь обратно в терминал

#### Шаг 5. Создай non-root пользователя

adduser your_name
usermod -aG sudo your_name
su - your_name
claude login

📋 Копировать

Снова авторизуйся через антидетект-браузер. Теперь Claude работает под безопасным пользователем.

#### Шаг 6. Подключи VS Code

1. В VS Code → Extensions → найди **Remote - SSH** (от Microsoft) → Install
2. `Cmd / Ctrl + Shift + P` → набери **Remote-SSH: Connect to Host**
3. Выбери **Add New SSH Host**
4. Введи: `your_name@IP_сервера`
5. Откроется новое окно VS Code - это сессия на сервере (внизу слева - зелёная полоска)

#### Шаг 7. Установи Claude Code на удалённое окно

1. В новом окне VS Code → Extensions
2. Найди **Claude Code** от Anthropic
3. Стрелка рядом с **Install** → **Install in SSH: IP_сервера**

**Готово.** Claude Code живёт на сервере в США, ты управляешь из VS Code на своём компе - без VPN.
